# 天地劫—万世仙宗 - 血脉系统设计

## 一、血脉系统概述

### 1.1 设计理念

血脉系统是游戏的核心养成系统之一，通过血脉的传承，为角色提供独特的属性加成和特性，增强游戏的策略性和可玩性。血脉系统与角色属性系统、繁育系统紧密结合，形成完整的角色成长体系。

### 1.2 核心机制

- **血脉类型**：不同种族的血脉，根据地域分布设计
- **血脉纯度**：0-100%，影响血脉潜力效果强度
- **血脉潜力**：影响六维属性的成长潜力
- **血脉特性**：血脉专属特性，稀有血脉和传说血脉拥有
- **血脉传承**：通过繁育系统传承血脉给后代

## 三、血脉纯度

### 3.1 纯度等级

| 纯度范围 | 等级 | 效果 | 视觉表现 |
|----------|------|------|----------|
| 0-12.5% | 微弱 | 基础效果 | 无特殊表现 |
| 12.6-25% | 稀薄 | 小幅效果 | 轻微光晕 |
| 25.1-37.5% | 中等 | 明显效果 | 明显光晕 |
| 37.6-50% | 浓郁 | 强力效果 | 强烈光晕 |
| 50.1-62.5% | 纯粹 | 极致效果 | 血脉特效显现 |
| 62.6-75% | 卓越 | 超越效果 | 血脉特效增强 |
| 75.1-87.5% | 完美 | 完美效果 | 血脉特效耀眼 |
| 87.6-100% | 极限 | 极限效果 | 血脉特效外放 |

### 3.2 纯度影响

- **属性加成**：纯度越高，血脉潜力对属性成长的影响越强
- **特性效果**：纯度越高，血脉特性效果越强
- **传承效果**：纯度越高，传承给后代的血脉比例越高

## 四、血脉潜力

### 4.1 潜力计算

**潜力计算公式**：
```
血脉潜力 = 基础潜力 × 纯度修正
```

**基础潜力**：每种血脉对六维属性的固有潜力值

**纯度修正**：
- 微弱：0.5
- 稀薄：0.6
- 中等：0.7
- 浓郁：0.8
- 纯粹：0.9
- 卓越：1.0
- 完美：1.1
- 极限：1.2

## 六、血脉传承

### 6.1 传承机制

**血脉池计算**：
- 父亲血脉比例 × 2
- 母亲血脉比例 × 2
- 两者相加得出孩子的血脉池

**血脉抽取**：
- 以12.5%为最小血脉单位
- 从血脉池中按照血脉占比权重抽取
- 每次抽取12.5%的一份血脉
- 重复抽取直到达到100%

**计算示例**：
```
父亲血脉：12.5%凤凰血脉，87.5%的人族血脉
母亲血脉：25%龙族血脉，75%的人族血脉

父亲血脉×2：25%凤凰血脉，175%的人族血脉
母亲血脉×2：50%龙族血脉，150%的人族血脉

血脉池合计：
- 凤凰血脉：25%
- 龙族血脉：50%
- 人族血脉：325%

按照权重抽取（每份12.5%）：
总份额 = 100% ÷ 12.5% = 8份

抽取结果（示例）：
- 人族血脉：50%（4份）
- 龙族血脉：37.5%（3份）
- 凤凰血脉：12.5%（1份）

孩子最终血脉构成：50%人族，37.5%龙族，12.5%凤凰
```

### 6.2 特性传承机制

**特性遗传**：
- 父母拥有的特性有50%概率遗传给孩子
- 多个特性独立计算遗传概率
- 遗传的特性会根据孩子的血脉纯度自动解锁对应等级

**无特性父母的特性解锁**：
- 当孩子血脉纯度达到对应特性要求时，有概率解锁特性
- 解锁概率：
  - 25%纯度：10%
  - 50%纯度：15%
  - 75%纯度：20%
  - 87.5%纯度：25%
  - 100%纯度：30%

**特性共存**：
- 所有特性可以同时生效
- 每个特性的外观效果从不同部位表现，避免冲突
- 高纯度特性的外观效果会覆盖低纯度特性的同类效果

### 6.3 传承优化

- **血脉融合**：多种血脉融合可能产生新的血脉
- **纯度遗传**：父母血脉纯度越高，传承效果越好
- **特殊传承**：特定血脉组合有特殊传承效果

## 九、血脉系统平衡

### 9.1 平衡性设计

- **血脉强度**：不同血脉各有特色，没有绝对强势的血脉
- **获取难度**：稀有血脉获取难度更高
- **成长曲线**：血脉成长需要时间和资源投入
- **团队搭配**：鼓励不同血脉的角色组成团队

### 9.2 稀有度平衡

- **普通血脉**：获取容易，成长稳定，无特性
- **稀有血脉**：获取困难，成长潜力大，2-3条特性
- **传说血脉**：获取极难，成长潜力极大，4-5条特性

这个血脉系统设计为游戏提供了以下特色：

1. **多样性**：根据地域分布设计多种血脉类型
2. **深度**：纯度、潜力、特性多层次养成
3. **策略性**：不同血脉搭配产生不同效果
4. **传承性**：通过繁育系统延续血脉，形成家族特色
5. **视觉表现**：血脉纯度的视觉效果
6. **平衡性**：各血脉各有特色，相互补充

血脉系统将成为游戏的核心玩法之一，为玩家提供丰富的养成体验和策略选择。
