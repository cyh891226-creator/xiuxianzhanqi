# Figma提示词 - 圣女系统

## 模块概述
圣女系统是游戏的核心剧情模块，包含圣女培养、圣地管理、圣女传承等功能。

---

```
创建一个修仙题材回合制战棋游戏的圣女培养主页界面。

1. 顶部导航栏（高度56px）：
   - 左侧：返回按钮
   - 中央：标题"圣女培养"
   - 右侧：圣地入口按钮

2. 圣女形象展示区（高度280px）：
   - 中央圣女大立绘（200x280）
   - 周围金色光芒粒子特效
   - 底部云雾缭绕效果

3. 圣女基本信息：
   - 圣女名称（艺术字体）
   - 身份标识：昆仑仙境继承人
   - 当前境界：金丹期

4. 培养进度面板：
   - 总进度条（百分比显示）
   - 声望数值：当前/目标
   - 培养阶段名称

5. 功能入口（4个卡片）：
   - 培养：提升圣女实力
   - 传承：圣女专属传承
   - 圣地：管理圣地事务
   - 羁绊：与圣女的羁绊故事

6. 当前任务提示：
   - 任务名称
   - 任务目标描述

设计风格：
- 界面：淡金色背景渐变
- 圣女立绘：金色边框 + 光晕
- 光芒特效：粒子动画效果
- 进度条：金色渐变填充
- 功能卡片：白色半透明 + 云纹

交互效果：
- 圣女立绘有轻微呼吸动画
- 光芒粒子持续飘动
- 点击卡片有选中高亮
- 进度提升有金光效果
```

## 2. 圣女培养界面（Saint Cultivation Interface）

### 提示词

```
创建一个修仙战棋游戏的圣女培养界面。

布局要求：
1. 圣女形象区：
   - 头像（96x96，圆形，金边）
   - 当前等级 + 培养进度

2. 可培养属性（4个方向）：
   - 实力：攻击/防御属性
   - 智慧：法术/策略属性
   - 魅力：影响力属性
   - 血脉：特殊血脉属性

3. 每个培养方向：
   - 属性图标 + 名称
   - 当前数值
   - 培养消耗
   - 培养按钮

4. 培养材料：
   - 材料图标列表
   - 拥有数量 / 需求数量
   - 获取途径按钮

5. 批量培养：
   - 一键培养按钮
   - 自动使用最优材料

6. 培养记录：
   - 最近培养历史
   - 属性变化显示

设计风格：
- 培养方向：四种颜色区分（红/蓝/金/紫）
- 培养按钮：对应颜色发光
- 材料充足：绿色数字
- 材料不足：红色数字
- 培养成功：对应属性金光

交互效果：
- 点击培养有属性上升动画
- 数值增加有跳动效果
- 培养成功有光芒扩散
- 材料不足有抖动提示
```
