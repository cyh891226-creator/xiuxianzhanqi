# 修仙战棋 - 美术音效资源需求

## 一、美术风格定位

### 1.1 整体风格
- **风格**：东方玄幻水墨风 + 轻度二次元
- **色调**：以青、白、金为主色调，魔道场景使用暗红、黑色
- **氛围**：仙气飘渺，意境深远

### 1.2 参考作品
- 《鬼谷八荒》：修仙氛围
- 《太吾绘卷》：国风表现
- 《阴阳师》：角色设计
- 《火焰纹章》：战棋UI

## 二、角色美术

### 2.1 角色立绘

#### 品质分级
| 品质 | 分辨率 | 细节程度 | 数量需求 |
|------|--------|----------|----------|
| 主角 | 2K | 精细 | 7个职业×2性别=14个 |
| 重要NPC | 2K | 精细 | 20个 |
| 普通NPC | 1K | 标准 | 50个 |
| 敌人 | 1K | 标准 | 30个 |
| BOSS | 2K | 精细 | 15个 |

#### 立绘要求
- **正面半身像**：用于对话、UI展示
- **战斗姿态**：用于战斗界面
- **表情差分**：喜怒哀乐等5种表情
- **境界变化**：高境界角色增加光效、服饰变化

### 2.2 角色Q版形象

#### 战斗用Q版
- **尺寸**：128×128像素
- **风格**：二头身Q版
- **动画**：
  - 待机：呼吸动画
  - 移动：行走/飞行动画
  - 攻击：普通攻击动画
  - 施法：技能施放动画
  - 受击：受伤动画
  - 死亡：倒下/消失动画

#### 数量需求
- 主角：14个
- 队友：30个
- 敌人：50个
- 召唤物：20个

### 2.3 角色设计要点

#### 剑修
- **特征**：背负长剑，白衣飘飘
- **颜色**：白色、青色、银色
- **元素**：剑、云、风

#### 丹修
- **特征**：手持药鼎，道袍宽袖
- **颜色**：绿色、棕色、金色
- **元素**：药草、丹炉、烟雾

#### 阵修
- **特征**：手持阵盘，星象环绕
- **颜色**：蓝色、紫色、银色
- **元素**：阵法、星图、符文

#### 体修
- **特征**：肌肉发达，僧袍/武服
- **颜色**：金色、棕色、红色
- **元素**：佛光、金刚、肌肉

#### 法修
- **特征**：手持法器，元素环绕
- **颜色**：根据元素变化
- **元素**：火、水、雷、冰

#### 血修
- **特征**：血色纹路，邪气凛然
- **颜色**：红色、黑色、暗红
- **元素**：血、魔气、暗影

#### 鬼修
- **特征**：阴气森森，鬼魂环绕
- **颜色**：紫色、黑色、青色
- **元素**：鬼魂、骷髅、幽火

## 四、UI美术

### 4.1 界面风格
- **风格**：国风水墨 + 现代扁平
- **色调**：根据界面功能变化
- **元素**：云纹、边框、水墨笔触

### 4.2 主要界面

#### 主界面
- **背景**：动态山水场景
- **按钮**：国风按钮，hover效果
- **信息栏**：角色头像、资源显示

#### 战斗界面
- **格子高亮**：移动范围、攻击范围
- **血条**：国风血条，带门派标识
- **技能栏**：圆形图标，冷却遮罩
- **信息面板**：角色详情、地形信息

#### 背包界面
- **格子**：物品格子，品质边框
- **分类标签**：武器、防具、饰品等
- **详情面板**：物品信息、装备对比

### 4.3 图标设计

#### 功能图标
- **尺寸**：64×64像素
- **风格**：线性图标 + 国风元素
- **数量**：50个

#### 技能图标
- **尺寸**：128×128像素
- **风格**：根据五行属性设计
- **数量**：200个

#### 物品图标
- **尺寸**：128×128像素
- **风格**：写实 + 游戏化
- **数量**：500个

### 4.4 字体设计

#### 中文字体
- **标题**：书法字体（如：方正清刻本悦宋）
- **正文**：清晰易读（如：思源黑体）
- **特效**：技能名称使用特效字体

#### 数字字体
- **伤害数字**：冲击力强的字体
- **资源数字**：清晰易读的字体

- **界面切换**： 淡入淡出 + 云纹

## 九、制作流程

### 9.1 美术制作流程
1. **概念设计**：草图、配色方案
2. **细化设计**：线稿、上色
3. **拆分制作**：角色拆分、场景分层
4. **动画制作**：Spine动画、特效制作
5. **导入测试**：游戏内测试、调整

### 9.2 音效制作流程
1. **需求确认**：音效列表、风格确认
2. **素材采集**：录制、购买素材
3. **音效制作**：剪辑、合成、处理
4. **音乐制作**：作曲、编曲、混音
5. **导入测试**：游戏内测试、调整

## 十一、总结

美术音效资源为修仙战棋提供了视听体验：

1. **美术风格**：东方玄幻水墨风，仙气飘渺
2. **角色设计**：7大职业，各有特色
3. **场景多样**：门派、野外、秘境多种场景
4. **UI设计**：国风元素，现代交互
5. **音效音乐**：传统乐器，东方韵味
6. **配音**：全角色配音，沉浸体验

这些资源将共同打造出一个沉浸式的修仙战棋世界。
