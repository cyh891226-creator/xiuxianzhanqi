# Figma提示词 - 主界面与导航系统

## 模块概述

主界面是玩家进入游戏后的第一个界面，包含所有核心系统的入口，需要清晰展示玩家状态和快速访问功能。根据最新系统设计，去掉了圣女系统，添加了炼丹和炼器入口。

### 提示词

```
创建一个修仙题材回合制战棋游戏的主界面设计。

布局要求：
1. 顶部状态栏（高度64px）：
   - 左侧：玩家头像（圆形，64x64），玩家名称（修仙风字体），境界图标+名称
   - 右侧：灵石数量图标+数值，仙玉数量图标+数值，设置齿轮图标，消息铃铛图标

2. 中央背景：动态仙侠风格背景（云雾飘动）

3. 主功能入口（4x2网格排列，每个按钮100x100px）：
   - 冒险：剑与山的图标，代表主线冒险
   - 伙伴：人物剪影图标，代表队伍管理
   - 宗门：宫殿建筑图标，代表宗门系统
   - 炼丹：丹炉图标，代表炼丹系统
   - 背包：储物袋图标，代表物品管理
   - 任务：卷轴图标，代表任务系统
   - 竞技场：擂台图标，代表PVP
   - 炼器：锤子与火焰图标，代表炼器系统

4. 底部状态栏：
   - 体力条（绿色渐变）+ 数值
   - 精力条（蓝色渐变）+ 数值
   - 每日任务快捷入口按钮

设计风格：
- 修仙古风元素：云纹、水墨边缘、仙鹤装饰
- 颜色：深青灰底色（#1a1a2e），金色强调（#ffd700），白色文字
- 按钮：带阴影的卡片式，hover时有光晕效果
- 字体：思源宋体/楷体风格

交互效果：
- 点击按钮有水波扩散动画
- 灵石/仙玉数字变化有跳动效果
- 新消息时铃铛有红色角标闪烁
```

## 2. 设置界面（Settings Panel）

### 提示词

```
创建一个修仙战棋游戏的设置面板（弹出式覆盖层）。

布局要求：
1. 整体为半透明黑色遮罩（rgba(0,0,0,0.7)）

2. 设置面板（居中，宽度480px，高度auto）：
   - 顶部标题栏：设置（修仙风字体）+ 关闭X按钮
   - 白色圆角背景（border-radius: 16px）

3. 设置分类标签（横向排列）：
   - 画面 | 声音 | 游戏 | 账户

4. 设置项列表（每个设置项高度56px）：
   - 画质设置：低/中/高/极致（滑动选择器）
   - 音乐音量：0-100滑动条
   - 音效音量：0-100滑动条
   - 自动战斗：开关toggle
   - 战斗速度：1x/1.5x/2x（按钮组）
   - 显示伤害数字：开关toggle
   - 显示辅助线：开关toggle

5. 底部按钮：
   - 恢复默认（次要按钮）
   - 保存并退出（主要按钮，金色）

设计风格：
- 设置面板带云纹边框装饰
- 滑动条：古风样式轨道，金色滑块
- Toggle开关：圆形，激活时发金光
- 按钮：圆角矩形，主要按钮金色渐变背景

交互效果：
- 面板从底部滑入
- 设置项hover时背景变浅金色
- 数值变化有数字跳动效果
```

## 3. 消息中心（Message Center）

### 提示词

```
创建一个修仙战棋游戏的消息中心界面（单机版）。

布局要求：
1. 整体为弹出面板（宽度400px，高度600px）

2. 顶部标签栏：
   - 全部 | 系统 | 任务 | 活动

3. 消息列表（可滚动）：
   每条消息（高度72px）：
   - 左侧：消息类型图标（40x40圆形）
   - 中间：消息标题 + 内容预览（单行截断）
   - 右侧：时间戳 + 未读红点
   - 底部：细线分隔

4. 空状态：
   - 中央：信封图标（灰色）
   - 文字：暂无消息

设计风格：
- 面板边缘云纹装饰
- 未读消息：左侧金色边条标识
- 时间戳：淡灰色小字
- 消息图标：根据类型区分（系统/任务/活动）

交互效果：
- 点击消息项有选中高亮
- 未读红点缓慢闪烁
- 已读消息自动标记
```
