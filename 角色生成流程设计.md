# 天地劫—万世仙宗 - 角色生成流程设计

## 1. 角色生成流程概述

角色生成是玩家进入游戏的第一个核心环节，通过一系列问题和选择来创建独特的游戏角色。角色生成与角色属性设计系统深度整合，确保每个选择都能影响角色的最终属性和潜力。

### 1.1 流程总览

```
性别选择 → 问心题环节 → 奇遇题环节 → 外貌定制 → 角色命名 → 属性确认
```

### 1.2 角色属性系统关联

| 生成环节 | 影响的属性系统 |
|----------|----------------|
| 问心题 | 精神品质特性、基础属性倾向 |
| 奇遇题 | 获得功法、属性加成、灵根类型、血脉构成 |
| 外貌定制 | 血脉外观特征（需与血脉系统联动） |
| 角色命名 | 无属性影响 |
| 属性确认 | 最终六维属性、战斗属性、潜力评价 |

## 2. 性别选择页面

### 2.1 功能需求

- 显示游戏背景故事简介
- 提供性别选择选项（男/女）
- 显示不同性别的预览形象
- 「下一步」按钮

### 2.2 设计元素

- 游戏背景图：隐世小村的水墨画风格
- 标题：「选择你的身份」
- 性别选项卡：左右布局，点击切换
- 预览窗口：显示所选性别的基础形象
- 背景故事文本：简洁描述游戏世界观
- 按钮样式：古风卷轴风格

### 2.3 交互流程

1. 玩家进入页面，默认选择男性
2. 点击性别选项切换预览形象
3. 点击「下一步」进入问心题环节

## 3. 问心题环节

### 3.1 设计目的

问心题环节旨在通过3道选择题影响角色的精神品质特性和基础属性倾向。每个选择对应不同的属性加成方向。

### 3.2 题目设计

#### 问心题一：面对危险时的第一反应

| 选项 | 属性倾向 | 特性方向 |
|------|----------|----------|
| A：挺身而出，保护他人 | 根骨+2，体魄+1 | 坚韧意志 |
| B：冷静分析，寻找最佳对策 | 神识+2，悟性+1 | 洞察先机 |
| C：暂时避开，保存实力 | 身法+2，气感+1 | 审时度势 |
| D：观察形势，伺机而动 | 神识+1，身法+2 | 静观其变 |

#### 问心题二：面对利益时的选择

| 选项 | 属性倾向 | 特性方向 |
|------|----------|----------|
| A：舍利取义，坚持原则 | 体魄+2，根骨+1 | 正气凛然 |
| B：权衡利弊，追求最大利益 | 悟性+2，神识+1 | 明智决策 |
| C：随波逐流，顺应形势 | 气感+2，身法+1 | 顺势而为 |
| D：无私奉献，帮助他人 | 神识+2，悟性+1 | 仁心仁术 |

#### 问心题三：面对困境时的态度

| 选项 | 属性倾向 | 特性方向 |
|------|----------|----------|
| A：坚持不懈，永不放弃 | 根骨+3 | 锲而不舍 |
| B：灵活变通，寻找新方法 | 悟性+2，身法+1 | 随机应变 |
| C：寻求帮助，团结协作 | 体魄+2，神识+1 | 同舟共济 |
| D：反思自我，提升能力 | 神识+2，气感+1 | 自我超越 |

### 3.3 精神品质特性

根据三道问心题的选择，生成对应的精神品质特性：

| 特性名称 | 效果 | 获取条件 |
|----------|------|----------|
| 勇者无惧 | 受到伤害-5% | 三题均选A |
| 智者千虑 | 经验获取+10% | 三题均选B |
| 隐者从容 | 闪避率+5% | 三题均选C |
| 仁者爱人 | 治疗效果+15% | 三题均选D |
| 中庸之道 | 全属性+1 | 混合选择 |

### 3.4 界面设计

- 背景：云雾缭绕的，问道台
- 问题文本：古风书法风格
- 选项按钮：竹简风格
- 进度指示器：显示当前是第几题
- 选项效果：悬停时发光效果

## 4. 奇遇题环节

### 4.1 设计目的

奇遇题环节通过5道选择题影响角色的获得功法、灵根类型、血脉构成和初始属性加成。

### 4.2 奇遇设计

#### 奇遇一：童年经历

**场景**：你在森林中迷路，遇到一位神秘老者

| 选项 | 影响结果 |
|------|----------|
| A：跟随老者学习医术 | 获得「木灵根」，初始悟性+3，获得《百草诀》 |
| B：向老者请教剑术 | 获得「金灵根」，初始根骨+3，获得《青锋剑谱》 |
| C：婉拒老者，自行寻找出路 | 获得「土灵根」，初始悟性+3，获得《阵图基础》 |

#### 奇遇二：少年时期的发现

**场景**：你在山洞中发现了一块发光的石头

| 选项 | 影响结果 |
|------|----------|
| A：伸手触摸石头 | 获得「雷灵根」，初始悟性+2，气感+2，获得《天雷诀》 |
| B：用工具挖掘带走 | 获得「土灵根」，初始体魄+3，获得《不动山决》 |
| C：记录位置后离开 | 获得「水灵根」，初始悟性+2，气感+2，获得《水云诀》 |

#### 奇遇三：成长过程的抉择

**场景**：你救治了一只受伤的神禽，它给了你一根羽毛

| 选项 | 影响结果 |
|------|----------|
| A：佩戴羽毛作为护身符 | 血脉：羽族血脉+25%，身法+2 |
| B：将其放生归山 | 血脉：妖族血脉+25%，体魄+2 |
| C：用它换取实用物资 | 获得「火灵根」，初始悟性+2，获得初始金币+100 |

#### 奇遇四：命运的考验

**场景**：村里爆发瘟疫，你也被感染

| 选项 | 影响结果 |
|------|----------|
| A：用草药自行治疗 | 获得「冰灵根」，体魄+3，免疫疾病 |
| B：寻求老者帮助 | 悟性+3，气感+2，获得额外特性「药理常识」 |
| C：依靠自身抵抗力硬抗 | 根骨+4，体魄+2，获得特性「百毒不侵」 |

#### 奇遇五：命运的转折

**场景**：仙人降临，要收你为徒

| 选项 | 影响结果 |
|------|----------|
| A：拜入仙家门下 | 神识+3，气感+3，血脉：神族血脉+12.5% |
| B：婉拒并继续修炼 | 身法+3，根骨+2，血脉：人族圣体+12.5% |
| C：请求仙人指点家人后离开 | 悟性+2，所有属性+1，获得道具「续命丹」×1 |

### 4.3 灵根系统整合

玩家通过奇遇题获得初始灵根，灵根品质随机生成：

| 灵根品质 | 纯度范围 | 潜力加成系数 |
|----------|----------|--------------|
| 凡品 | 41-60% | 1.0 |
| 良品 | 61-75% | 1.2 |
| 上品 | 76-90% | 1.5 |
| 极品 | 91-99% | 2.0 |
| 先天灵根 | 100% | 3.0 |

### 4.4 血脉系统整合

玩家通过奇遇题获得初始血脉构成，血脉纯度根据选择累积：

| 血脉类型 | 稀有度 | 奇遇获取方式 |
|----------|--------|--------------|
| 羽族血脉 | 稀有 | 救助神禽选项 |
| 妖族血脉 | 普通 | 放生选项 |
| 神族血脉 | 稀有 | 拜师选项 |
| 人族圣体 | 传说 | 婉拒仙人选项 |

### 4.5 界面设计

- 背景：时间流逝的动画效果
- 故事文本：卷轴展开效果
- 选项按钮：古纸风格
- 进度指示器：显示当前是第几题

## 7. 最终属性确认页面

### 7.1 角色属性计算

根据前述所有选择，计算角色的最终属性：

#### 六维基础属性计算

```
最终属性 = 基础值 + 问心题加成 + 奇遇加成
```

| 属性 | 基础值范围 | 说明 |
|------|------------|------|
| 根骨 | 1-10 | 肉身力量，影响物理攻击 |
| 体魄 | 1-10 | 体质根基，影响物理防御 |
| 悟性 | 1-10 | 法术天赋，影响法术攻击 |
| 气感 | 1-10 | 法力底蕴，影响法术防御 |
| 神识 | 1-10 | 洞察能力，影响命中暴击 |
| 身法 | 1-10 | 敏捷反应，影响闪避移动 |

#### 潜力计算

```
最终潜力值 = (血脉潜力 × 70% + 灵根潜力 × 30%)
```

| 潜力等级 | 最终潜力值范围 | 每级成长 | 属性上限 |
|----------|----------------|----------|----------|
| D | <1.0 | +0.5 | 40 |
| C | 1.0-1.9 | +1 | 60 |
| B | 2.0-2.9 | +1.5 | 80 |
| A | 3.0-3.9 | +2 | 100 |
| SS | 4.0-4.9 | +2.5 | 120 |
| SSS | ≥5.0 | +3 | 150 |

#### 战斗属性计算

| 属性 | 计算公式 |
|------|----------|
| 生命值 | 体魄 × 10 + 等级 × 体魄潜力成长 |
| 法力值 | 气感 × 8 + 等级 × 气感潜力成长 |
| 攻击力 | 根骨 × 2 + 武器加成 |
| 法术强度 | 悟性 × 2 + 装备加成 |
| 防御力 | 体魄 × 1.5 + 装备加成 |
| 法术抗性 | 气感 × 1.5 + 装备加成 |
| 速度 | 身法 × 1.2 + 装备加成 |
| 暴击率 | 神识 × 0.4 + 装备加成 |
| 闪避率 | 身法 × 0.4 + 装备加成 |
| 命中值 | 神识 × 0.4 + 装备加成 |

## 9. 设计风格统一要求

### 9.1 视觉风格

- **整体风格**：古风修仙风格，融合水墨画元素
- **色彩方案**：以青、白、金、红为主，体现修仙氛围
- **字体**：古风书法字体，清晰易读

### 9.2 动效设计

- 流畅的过渡动画，如卷轴展开、云雾缭绕
- 点击时的水墨扩散效果
- 选择时的光晕效果
- 属性数值变化的数字跳动效果

### 9.3 交互设计

- 支持Steam平台的鼠标操作
- 支持后续手机端的触摸操作
- 响应式设计，适配不同屏幕尺寸
