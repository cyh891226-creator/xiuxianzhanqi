# 天地劫—万世仙宗 - 角色属性设计

## 1. 属性系统概述

为了更符合仙侠题材的特点，我们重新设计了角色属性系统，将传统的六维属性转化为更具修仙特色的属性体系。

## 2. 基础属性

### 2.1 核心属性

| 属性名称 | 描述 | 影响效果 |
|----------|------|----------|
| 根骨 | 肉身力量 | 物理攻击、格挡减伤、物理暴击伤害 |
| 体魄 | 体质根基 | 物理防御、格挡概率、抗暴 |
| 悟性 | 法术天赋 | 法术攻击、法术抵抗、法术暴击伤害 |
| 气感 | 法力底蕴 | 法术防御、法术抵抗、法力上限、能量回复速率 |
| 神识 | 洞察能力 | 命中、暴击率（通用） |
| 身法 | 敏捷反应 | 闪避、移动力 |

### 2.2 属性潜力系统

每个基础属性都有对应的潜力值，影响属性成长上限：
- **SSS级潜力**：属性上限150，每级成长+3
- **SS级潜力**：属性上限120，每级成长+2.5
- **A级潜力**：属性上限100，每级成长+2
- **B级潜力**：属性上限80，每级成长+1.5
- **C级潜力**：属性上限60，每级成长+1
- **D级潜力**：属性上限40，每级成长+0.5

## 3. 战斗属性

### 3.1 基本战斗属性

| 属性名称 | 描述 | 计算方式 |
|----------|------|----------|
| 生命值 | 角色的生命总量 | 体魄 × 10 + 装备加成 |
| 法力值 | 角色的法力总量 | 气感 × 8 + 装备加成 |
| 攻击力 | 物理攻击能力 | 根骨 × 2 + 武器加成 |
| 法术强度 | 法术攻击能力 | 悟性 × 2 + 装备加成 |
| 防御力 | 物理防御能力 | 体魄 × 1.5 + 装备加成 |
| 法术抗性 | 法术防御能力 | 气感 × 1.5 + 装备加成 |
| 速度 | 移动力和反应速度，影响反击机会 | 身法 × 1.2 + 装备加成 |
| 移动力 | 每回合可移动的格数 | 速度值决定 |
| 暴击率 | 暴击触发概率 | 神识 × 0.4 + 装备加成 |
| 暴击伤害 | 暴击时的伤害倍数 | 基础150% + 根骨 × 0.5%（物理）/ 悟性 × 0.5%（法术） |
| 闪避率 | 躲避攻击的概率 | 身法 × 0.4 + 装备加成 |
| 命中值 | 攻击命中的概率 | 神识 × 0.4 + 装备加成 |
| 生命恢复 | 每回合生命值恢复 | 体魄 × 0.5 + 装备加成 |
| 法力恢复 | 每回合法力值恢复 | 气感 × 0.8 + 装备加成 |

### 3.2 速度与移动力机制

#### 移动力计算
- **基础移动力**：速度值 ÷ 10（向下取整）
- **最小移动力**：2格
- **最大移动力**：10格

#### 行动消耗
- **移动消耗**：每移动1格消耗1点移动力
- **第一次攻击/技能**：免费，不消耗移动力
- **第二次攻击/技能**：消耗2点移动力
- **第三次攻击/技能**：消耗3点移动力
- **技能消耗**：根据技能等级消耗2-5点移动力

#### 多攻击机制
- 如果剩余移动力 ≥ 攻击消耗，可以再次攻击
- 每回合最多3次攻击或技能
- 第一次攻击或技能免费，后续攻击/技能消耗移动力

#### 反击机制
- 每回合首次受击未死亡时，必定进行1次反击
- 反击为全额伤害（100%伤害）
- **先手反击**：速度差>50%时，受击方有50%概率抢先反击（可先于攻击方行动）
- **必先反击**：速度差>100%时，受击方必定抢先反击
- 反击不消耗移动力

## 4. 修为境界系统

### 4.1 境界划分

每个大境界分为10个小境界（例如：练气期一重至练气期十重）。

| 大境界名称 | 小境界范围 | 特殊能力 |
|------------|------------|----------|
| 练气期 | 1-10重 | 能够感应天地灵气 |
| 筑基期 | 11-20重 | 能够存储灵气，使用基础法术 |
| 金丹期 | 21-30重 | 凝结金丹，寿元增加 |
| 元婴期 | 31-40重 | 诞生元婴，灵魂离体 |
| 化神期 | 41-50重 | 神念外放，隔空取物 |
| 炼虚期 | 51-60重 | 虚实转换，空间穿梭 |
| 合体期 | 61-70重 | 身心合一，天人感应 |
| 大乘期 | 71-80重 | 掌握法则，呼风唤雨 |
| 真仙期 | 81-90重 | 超脱凡俗，寿元无尽 |
| 大道期 | 91-100重 | 掌控大道，与天同齐 |

### 4.2 境界属性压制

大境界之间的属性压制规则：
- 高一个大境界的角色对低境界角色有10%属性压制
- 属性压制影响伤害计算：攻击高境界目标时，伤害降低10%
- 属性压制影响防御：高境界角色受到低境界攻击时，伤害提升10%

### 4.3 境界突破机制

#### 小境界突破（升级）
- **突破条件**：经验值达到当前小境界上限
- **突破方式**：自动突破，无需消耗物品
- **属性提升**：根据六维属性潜力值增加对应属性

#### 大境界突破
- **突破条件**：达到当前大境界第10重，并拥有突破丹药
- **突破方式**：使用突破丹药进行突破
- **丹药品质影响**：
  - 下品突破丹：突破成功率60%，获得10%基础属性加成
  - 中品突破丹：突破成功率75%，获得15%基础属性加成
  - 上品突破丹：突破成功率90%，获得18%基础属性加成
  - 极品突破丹：突破成功率100%，获得20%基础属性加成
  - 神品突破丹：突破成功率100%，获得25%基础属性加成，额外解锁一个被动技能槽

#### 突破结果与特性
- **突破成功**：属性大幅提升，获得新的被动技能槽，有概率获得正面特性
  - 极品突破成功：30%概率获得「天人交感」特性（气感潜力+1级）
  - 普通突破成功：10%概率获得「根基稳固」特性（全属性潜力+0.5级）

- **突破失败**：修为倒退，属性损失，有概率获得负面特性
  - 突破失败：50%概率获得「根基损伤」特性（六维属性潜力各-1级）
  - 突破失败：30%概率获得「心魔入侵」特性（修炼经验获取-20%）

## 8. 属性成长系统

### 8.1 升级属性提升

- **每次小境界升级**：根据角色六维属性潜力值增加对应属性
  - SSS级潜力：每级+3
  - SS级潜力：每级+2.5
  - A级潜力：每级+2
  - B级潜力：每级+1.5
  - C级潜力：每级+1
  - D级潜力：每级+0.5
- **大境界突破**：获得额外的属性百分比加成（根据突破丹药品质）
- **修炼功法**：不同功法提供不同的属性成长倾向

### 8.2 功法对属性的影响

| 功法类型 | 主要属性加成 | 次要属性加成 |
|----------|--------------|--------------|
| 力量型 | 根骨+3/级 | 体魄+1/级 |
| 敏捷型 | 身法+3/级 | 根骨+1/级 |
| 智力型 | 悟性+3/级 | 神识+1/级 |
| 体质型 | 体魄+3/级 | 根骨+1/级 |
| 精神型 | 神识+3/级 | 悟性+1/级 |
| 平衡型 | 所有属性+1/级 | - |

### 8.3 装备对属性的影响

- **武器**：主要提升攻击力和根骨
- **防具**：主要提升防御力和体魄
- **饰品**：提供各种特殊属性加成
- **套装**：集齐套装获得额外属性加成

## 9. 属性与技能的关系

### 9.1 技能学习条件

- **基础技能**：对应属性达到10点
- **中级技能**：对应属性达到20点
- **高级技能**：对应属性达到30点
- **绝学**：对应属性达到40点，特定条件
- **仙术**：对应属性达到50点，特殊机缘

### 9.2 技能威力计算

- **物理技能**：威力 = 技能基础威力 × (1 + 根骨 × 0.02)
- **法术技能**：威力 = 技能基础威力 × (1 + 悟性 × 0.02) × 灵根加成
- **辅助技能**：效果 = 技能基础效果 × (1 + 气感 × 0.02)
- **控制技能**：成功率 = 基础成功率 + 神识 × 0.01 - 目标神识 × 0.005

### 9.3 灵根对元素技能加成

拥有对应灵根时，该元素技能威力获得额外加成：
- **极品灵根**：元素技能威力+100%
- **上品灵根**：元素技能威力+50%
- **良品灵根**：元素技能威力+20%
- **凡品灵根**：元素技能威力+0%
- **无灵根**：无法学习和使用该元素技能

## 11. 游戏内实现建议

### 11.1 属性显示

- **主界面**：显示核心属性和当前状态
- **详细面板**：显示所有属性的详细数值和评价
- **升级界面**：显示属性加点选项和预览
- **装备界面**：显示装备对属性的影响
- **潜力界面**：显示各属性潜力等级和成长值

### 11.2 属性计算

- **实时计算**：装备和状态变化时实时更新属性
- **潜力计算**：根据潜力等级计算属性成长
- **境界压制**：战斗时计算属性压制效果
- **缓存优化**：使用缓存减少计算量
- **显示简化**：对玩家显示关键属性，隐藏复杂计算

### 11.3 属性平衡

- **职业平衡**：确保不同职业的属性体系平衡
- **潜力平衡**：确保不同潜力等级的差异合理
- **境界平衡**：确保境界提升带来合理的属性增长
- **装备平衡**：确保装备属性加成合理，避免数值膨胀
- **丹药平衡**：确保不同品质丹药的价值差异合理

### 11.4 扩展性

- **预留扩展位**：为未来的属性系统扩展预留空间
- **新特性系统**：支持添加新的特性类型
- **新血脉系统**：支持添加新的血脉类型
- **新灵根系统**：支持添加新的灵根类型
- **mod支持**：支持玩家自定义属性系统
- **DLC兼容**：确保新内容与现有属性系统兼容

## 12. 与其他系统的关联

### 12.1 与角色生成的关联

- **问心题**：影响精神品质特性和基础属性
- **奇遇题**：影响初始职业、属性加成和血脉构成
- **外貌定制**：与血脉构成相关联
- **初始潜力**：根据选择确定各属性潜力等级

### 12.2 与宗门系统的关联

- **宗门建筑**：提供属性加成和修炼效率提升
- **宗门科技**：解锁属性提升途径和潜力突破
- **宗门职位**：提供职位属性加成
- **宗门功法**：提供额外的属性成长

### 12.3 与战斗系统的关联

- **属性影响**：直接影响战斗能力
- **技能释放**：属性决定技能效果和威力
- **战斗策略**：基于属性和潜力制定战斗策略
- **境界压制**：高境界对低境界的属性压制效果

### 12.4 与丹药系统的关联

- **突破丹药**：用于大境界突破，影响突破成功率和属性加成
- **属性丹药**：直接提升属性值
- **潜力丹药**：提升属性潜力等级
- **特殊丹药**：获得或移除特性

### 12.5 与剧情系统的关联

- **圣女**：作为特殊剧情角色出现在游戏中，不影响角色属性系统
- **剧情任务**：完成特定剧情任务可获得属性丹药和特性
- **奇遇系统**：特定奇遇可提升属性或获得特殊特性
