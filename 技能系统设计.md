# 天地劫—万世仙宗 - 技能系统设计

## 一、技能体系概述

### 1.1 设计理念
技能系统基于功法体系构建，通过功法提供属性加成和技能，角色修炼不同类型的功法成为不同的职业。功法本身具有修炼境界，通过战斗和修炼提升境界，获得更强的属性加成和技能效果。

### 1.2 核心机制
- **功法决定职业**：角色修炼的功法类型决定角色职业（如剑道功法→剑修）
- **功法提供技能**：功法包含多种技能，角色通过修炼获得
- **属性解锁技能**：角色升级时若属性达到技能解锁所需属性值，则有概率获得或提升技能
- **多功法修炼**：角色可同时修炼多种功法，但只能装备一种功法获取属性加成
- **技能携带限制**：每个角色可携带1个主动技能、1个受击型技能、3个战斗型被动、4个属性型被动

## 三、功法修炼境界系统

### 3.1 功法境界划分

每种功法都有独立的修炼境界，从低到高分为：

| 境界名称 | 境界等级 | 属性加成比例 | 解锁技能比例 | 说明 |
|----------|----------|--------------|--------------|------|
| 入门 | 1 | 20% | 30% | 初步掌握功法 |
| 小成 | 2 | 40% | 50% | 基本掌握功法要领 |
| 大成 | 3 | 60% | 70% | 完全掌握功法精髓 |
| 圆满 | 4 | 80% | 90% | 功法运用自如 |
| 大圆满 | 5 | 100% | 100% | 功法修炼极致 |

### 3.2 功法境界经验

功法境界经验通过以下方式获取：

| 获取方式 | 经验值 | 说明 |
|----------|--------|------|
| 战斗胜利 | 10-30 | 根据敌人难度浮动 |
| 战斗失败 | 5-15 | 根据敌人难度浮动 |
| 闭关修炼 | 50-100 | 根据闭关时间浮动 |
| 特殊训练 | 30-50 | 完成宗门训练任务 |
| 顿悟 | 100-200 | 特殊事件触发 |

### 3.3 功法境界升级条件

| 当前境界 | 升级所需经验 | 升级成功率 |
|----------|--------------|------------|
| 入门→小成 | 100 | 100% |
| 小成→大成 | 200 | 100% |
| 大成→圆满 | 400 | 80% |
| 圆满→大圆满 | 800 | 50% |

### 3.4 功法境界属性计算

装备功法的最终属性加成计算：

**实际属性加成 = 功法基础属性加成 × 功法品质系数 × 功法境界加成比例**

## 四、功法与职业系统

### 4.1 功法类型与职业对应

| 功法类型 | 职业名称 | 角色标签 | 主要属性 | 核心特点 |
|----------|----------|----------|----------|----------|
| 剑道功法 | 剑修 | 剑 | 根骨、身法 | 高爆发、高暴击、远程攻击 |
| 体道功法 | 体修 | 体 | 体魄、根骨 | 高防御、高生命、反击 |
| 医道功法 | 医修 | 医 | 神识、体魄 | 治疗、净化、辅助 |
| 阵道功法 | 阵法师 | 阵 | 悟性、神识 | 伤害、控制、范围效果 |
| 法道功法 | 法修 | 法 | 悟性、气感 | 元素攻击、范围伤害、控制 |
| 魔道功法 | 魔修 | 魔 | 根骨、气感 | 吸血、爆发、诅咒 |
| 儒道功法 | 儒修 | 儒 | 神识、悟性 | 增益、削弱、辅助 |
| 佛道功法 | 佛修 | 佛 | 体魄、神识 | 防护、净化、治疗 |

### 4.2 功法学习条件

- **灵根要求**：必须拥有对应灵根才能学习该元素功法
- **属性要求**：主要属性达到功法要求值
- **境界要求**：达到功法要求的最低境界
- **前置功法**：部分高级功法需要前置功法基础

### 4.3 多功法修炼机制

#### 4.3.1 功法修炼规则
- **可修炼数量**：每个角色最多同时修炼3种功法
- **装备限制**：只能装备1种功法，其他功法作为储备
- **属性获取**：只有装备的功法提供属性加成和技能
- **境界独立**：每种功法有独立的修炼境界

#### 4.3.2 功法切换
- **切换消耗**：切换装备功法消耗24小时冷却时间
- **冷却机制**：切换后原功法境界保留，新功法境界不变
- **技能保留**：已获得的技能保留，但只有装备功法技能可携带

#### 4.3.3 储备功法管理
- **经验获取**：储备功法可继续通过战斗获取经验
- **境界提升**：储备功法境界可正常提升
- **技能解锁**：储备功法技能解锁后，可在切换后携带

## 六、技能分类系统

### 6.1 技能分类

技能分为四大类：

#### 6.1.1 主动技能
- **定义**：需要主动释放的技能，消耗法力
- **特点**：威力大，效果多样，需要策略性使用
- **消耗**：消耗法力值，根据技能威力不同消耗不同
- **携带限制**：每个角色最多携带1个主动技能

**示例**：
- **御剑术**：远程攻击敌人单体，造成150%攻击力的物理伤害，消耗20法力
- **火球术**：远程攻击敌人单体，造成120%法术强度的火属性伤害，消耗15法力
- **治疗术**：治疗友方单体，恢复200%法术强度的生命值，消耗25法力

#### 6.1.2 受击型技能
- **定义**：受到攻击时自动触发的技能，消耗法力
- **特点**：被动触发，提供防御或反击效果
- **消耗**：触发时消耗法力值，根据技能效果不同消耗不同
- **携带限制**：每个角色最多携带1个受击型技能

**示例**：
- **先天罡气**：受击时减少30%受到的伤害，同时反弹20%伤害给敌人，消耗10法力
- **护体罡气**：受击时有50%概率格挡攻击，减少50%伤害，消耗8法力
- **反震护盾**：受击时反弹50%伤害给敌人，消耗15法力

#### 6.1.3 战斗型被动
- **定义**：战斗中持续生效的被动技能，无消耗
- **特点**：自动生效，提供持续的战斗增益
- **消耗**：无消耗
- **携带限制**：每个角色最多携带3个战斗型被动

**示例**：
- **五气朝元**：每回合开始时回复10%法力值和5%生命值
- **战斗本能**：每回合首次攻击伤害+20%
- **坚韧意志**：生命值低于30%时，受到的伤害减少30%

#### 6.1.4 属性型被动
- **定义**：永久提升角色属性的被动技能，无消耗
- **特点**：永久生效，直接提升角色属性
- **消耗**：无消耗
- **携带限制**：每个角色最多携带4个属性型被动

**示例**：
- **迅捷**：速度+10
- **强壮**：根骨+8
- **聪慧**：悟性+8
- **坚韧**：体魄+8

### 6.2 技能携带限制

每个角色的技能携带限制：

| 技能类型 | 最大携带数量 | 说明 |
|----------|--------------|----------|
| 主动技能 | 1 | 战斗中主动释放，消耗法力 |
| 受击型技能 | 1 | 受击时触发，消耗法力 |
| 战斗型被动 | 3 | 战斗中持续生效，无消耗 |
| 属性型被动 | 4 | 永久提升属性，无消耗 |
| **总计** | **9** | - |

### 6.3 技能切换机制

- **战斗外切换**：战斗外可自由切换已获得的技能
- **战斗内切换**：战斗中无法切换技能
- **技能预设**：可预设多套技能配置，战斗前快速切换
- **自动配置**：系统可根据角色属性和功法自动推荐技能配置

## 七、技能获取与升级系统

### 7.1 技能获取方式

技能通过以下方式获取：

| 获取方式 | 获取内容 | 能否提升等级 | 说明 |
|----------|----------|--------------|------|
| 角色升级 | 概率获得新技能或提升技能等级 | 可以 | 根据功法品质和属性条件判定 |
| 闭关修炼 | 提升已装备功法的技能等级 | 可以 | 消耗时间和资源 |
| 角色传授 | 解锁技能（不提升等级） | 否 | 角色间互相传授已解锁技能 |

### 7.2 技能解锁条件

技能通过以下条件解锁：

| 解锁条件 | 说明 | 示例 |
|----------|------|------|
| 属性要求 | 对应属性达到指定值 | 根骨≥20解锁御剑术 |
| 境界要求 | 功法境界达到指定等级 | 小成境界解锁进阶技能 |
| 功法等级 | 功法修炼达到指定等级 | 功法等级5解锁新技能 |
| 特殊条件 | 完成特定任务或事件 | 完成试炼任务解锁特殊技能 |

### 7.3 技能获取概率

角色升级时，若属性达到技能解锁所需属性值，则有概率获得或提升技能：

| 天中 | 70% | 60% | |

| 地中 | 45% | 40% | |
| 地下 | 40% | 35% | |

### 7.4 技能获取流程

1. **角色升级**：角色获得经验值升级
2. **检查属性**：系统检查角色属性是否达到技能解锁条件
3. **概率判定**：根据功法品质进行概率判定
4. **技能获取**：判定成功则获得新技能或提升已有技能等级
5. **技能提示**：向玩家显示技能获取或提升的提示

### 7.5 闭关修炼系统

#### 7.5.1 闭关修炼条件
- **装备功法**：必须装备要修炼技能的功法
- **时间消耗**：每次闭关消耗12小时
- **资源消耗**：消耗一定量的灵石或丹药

#### 7.5.2 闭关修炼效果
- **技能提升**：提升装备功法中指定技能1级
- **成功率**：根据技能等级和功法品质，成功率60%-95%
- **失败惩罚**：失败后返还50%资源，技能不提升

#### 7.5.3 闭关修炼限制
- **每日次数**：每日最多进行2次闭关修炼
- **冷却时间**：每次闭关后需间隔6小时才能再次闭关

### 7.6 角色传授系统

#### 7.6.1 传授条件
- **角色等级**：传授角色等级≥30级
- **技能状态**：传授的技能必须已解锁且等级≥3
- **好感度**：角色间好感度需达到"友好"以上

#### 7.6.2 传授效果
- **技能解锁**：被传授角色获得该技能（等级为1）
- **等级限制**：通过传授获得的技能无法通过任何方式提升等级
- **技能标记**：传授获得的技能显示特殊标记"授"

#### 7.6.3 传授限制
- **次数限制**：每个角色最多传授3次技能
- **技能限制**：每个角色最多通过传授获得5个技能
- **品质限制**：高品质功法的核心技能不可传授

## 十、技能平衡设计

### 10.1 伤害公式

```
技能伤害 = (攻击者属性 × 技能威力系数 - 防御者属性) × 克制系数 × 随机系数

随机系数：0.9 - 1.1
克制系数：1.5（克制）/ 1.0（普通）/ 0.7（被克制）
```

### 10.2 消耗设计

| 技能类型 | 法力消耗 | 冷却时间 | 说明 |
|----------|----------|----------|------|
| 低阶主动技能 | 10-20 | 1-2回合 | 基础技能，频繁使用 |
| 中阶主动技能 | 25-40 | 3-4回合 | 进阶技能，策略使用 |
| 高阶主动技能 | 45-60 | 5-6回合 | 高级技能，关键时刻使用 |
| 受击型技能 | 5-15 | 无冷却 | 被动触发，消耗较低 |
| 战斗型被动 | 无 | 无 | 持续生效，无消耗 |
| 属性型被动 | 无 | 无 | 永久生效，无消耗 |

### 10.3 平衡原则

1. **品质差异**：高品质功法技能威力强，但获取难度大
2. **职业互补**：没有全能职业，需要团队配合
3. **克制关系**：五行相克影响技能效果
4. **成长曲线**：技能随等级提升而增强
5. **策略深度**：技能携带限制迫使玩家做出选择

## 十二、总结

技能系统基于功法体系构建，具有以下特点：

1. **职业精简**：保留剑修、体修、医修、阵法师、法修、魔修、儒修、佛修八种职业
2. **功法品质分明**：天地玄黄四大品质十二个等级，差异明显
3. **功法境界系统**：入门到 大圆满五层境界，通过战斗和修炼提升
4. **多功法修炼**：可同时修炼3种功法，但只能装备1种获取属性
5. **技能分类清晰**：主动、受击型、战斗型被动、属性型被动四大类
6. **携带限制合理**：每个角色最多携带9个技能
7. **获取方式多样**：角色升级概率获取、闭关修炼提升、角色传授解锁
8. **传授技能限制**：传授技能不能提升等级，增加策略深度

这个技能系统为游戏提供了丰富的战斗策略和深度养成体验。
