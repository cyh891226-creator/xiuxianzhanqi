# Figma提示词 - 炼丹系统

## 模块概述

炼丹系统是游戏的核心养成系统之一，通过收集材料、炼制丹药，为玩家提供属性提升、境界突破、战斗辅助等功能。炼丹界面需要展示丹炉、丹方、材料和炼丹师等信息。

---

### 提示词

```
创建一个修仙战棋游戏的炼丹系统主界面设计。

布局要求：
1. 顶部导航栏（高度56px）：
   - 左侧：返回按钮（箭头图标）
   - 中央：炼丹房标题（修仙风字体）
   - 右侧：当前炼丹师信息（头像+等级）

2. 丹炉区域（占据主要空间）：
   - 丹炉图标展示（120x120px，动态火焰效果）
   - 下方：选择丹方下拉框
   - 下方：开始炼制按钮

3. 功能面板（左右排列）：
   - 左侧：材料背包按钮（显示材料数量角标）
   - 右侧：丹方列表按钮

4. 底部：炼丹师信息卡片
   - 炼丹师头像
   - 炼丹师等级
   - 成功率加成显示

设计风格：
- 整体背景：丹炉房风格，有火焰光效
- 丹炉：带动态火焰动画，品质越高火焰越绚烂
- 按钮：圆角矩形，炼丹相关图标
- 面板：半透明深色背景

交互效果：
- 丹炉有呼吸光效
- 选择丹方后显示所需材料
- 开始炼制有火焰聚集动画
- 炼制完成有丹药出现动画
```

```
创建一个修仙战棋游戏的丹方列表界面设计。

   - 左侧：返回按钮

- 列表项：带淡淡边框

交互效果：
- 点击丹方展开详情
- 品质标签有微光效果
- 滚动时有惯性效果
```

### 提示词

```
创建一个修仙战棋游戏的丹药炼制界面设计。

布局要求：
1. 顶部导航栏：
   - 左侧：返回按钮
   - 中央：炼制状态（炼制中/炼制完成）

2. 丹炉展示区（主要空间）：
   - 大型丹炉图标（200x200px）
   - 动态火焰动画效果
   - 炼制进度条（百分比显示）

3. 材料显示区：
   - 显示所需材料及数量
   - 每个材料显示：图标 + 名称 + 数量
   - 已放入显示对勾标识
   - 缺少材料显示红色标识

4. 炼制信息：
   - 预计获得物品
   - 当前成功率百分比

5. 底部按钮：
   - 取消炼制（次要按钮）

设计风格：
- 丹炉火焰：根据炼制进度变化颜色
- 进度条：金色填充，云纹装饰
- 材料：图标+文字组合，绿色对勾/红色叉号

交互效果：
- 炼制中有火焰跳动动画
- 进度条平滑增长
- 完成时有丹药闪光特效
- 成功/失败有不同动画反馈
```
