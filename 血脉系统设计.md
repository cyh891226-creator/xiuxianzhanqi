# 天地劫—万世仙宗 - 血脉系统设计

## 一、血脉系统概述

### 1.1 设计理念

血脉系统是游戏的核心养成系统之一，通过血脉的传承、觉醒和进化，为角色提供独特的属性加成和技能，增强游戏的策略性和可玩性。血脉系统与角色属性系统、繁育系统、技能系统紧密结合，形成完整的角色成长体系。

### 1.2 核心机制

- **血脉类型**：不同种族的血脉，如人族、龙族、凤凰等
- **血脉纯度**：0-100%，影响血脉效果强度
- **血脉潜力**：影响对应属性的成长潜力
- **血脉技能**：血脉专属技能，随纯度和等级解锁
- **血脉觉醒**：特定条件下血脉觉醒，获得强大能力
- **血脉进化**：通过特殊材料和仪式进化血脉
- **血脉传承**：通过繁育系统传承血脉给后代

## 三、血脉纯度

### 3.1 纯度等级

| 纯度范围 | 等级 | 效果 | 视觉表现 |
|----------|------|------|----------|
| 0-20% | 微弱 | 基础效果 | 无特殊表现 |
| 21-40% | 稀薄 | 小幅效果 | 轻微光晕 |
| 41-60% | 中等 | 明显效果 | 明显光晕 |
| 61-80% | 浓郁 | 强力效果 | 强烈光晕 |
| 81-100% | 纯粹 | 极致效果 | 血脉特效显现 |

### 3.2 纯度影响

- **属性加成**：纯度越高，属性加成越强
- **技能效果**：纯度越高，血脉技能效果越强
- **觉醒概率**：纯度越高，觉醒概率越大
- **进化成功率**：纯度越高，进化成功率越高
- **传承效果**：纯度越高，传承给后代的血脉比例越高

### 3.3 纯度提升

**提升途径**：
- **血脉提纯丹**：直接提升血脉纯度
- **特殊材料**：使用对应血脉的特殊材料
- **血脉仪式**：通过特定仪式提升纯度
- **血脉觉醒**：觉醒后纯度大幅提升
- **进化**：进化后纯度重置为100%

**提升限制**：
- 每次提升有成功率
- 纯度越高，提升难度越大
- 不同血脉有不同的提升上限

## 四、血脉潜力

### 4.1 潜力计算

**潜力计算公式**：
```
血脉潜力 = 基础潜力 × 纯度修正
```

**基础潜力**：每种血脉的固有潜力值

| 血脉名称 | 基础潜力 |
|----------|----------|
| 人族血脉 | 0.8 |
| 龙族血脉 | 1.2 |
| 凤凰血脉 | 1.1 |
| 玄武血脉 | 1.3 |
| 白虎血脉 | 1.2 |
| 朱雀血脉 | 1.1 |
| 麒麟血脉 | 1.0 |
| 混沌血脉 | 1.5 |

**纯度修正**：
- 微弱：0.5
- 稀薄：0.7
- 中等：0.9
- 浓郁：1.1
- 纯粹：1.3

### 4.2 潜力影响

- **属性成长**：潜力越高，每级属性成长越多
- **技能学习**：潜力越高，技能学习速度越快
- **突破成功率**：潜力越高，境界突破成功率越高
- **血脉觉醒**：潜力越高，觉醒效果越强

## 六、血脉觉醒

### 6.1 觉醒条件

- **血脉纯度**：达到80%以上
- **境界要求**：达到化神期以上
- **觉醒材料**：对应血脉的觉醒材料
- **觉醒仪式**：在特定地点进行觉醒仪式
- **随机因素**：有一定的觉醒概率

### 6.2 觉醒效果

- **纯度提升**：觉醒后血脉纯度提升至90%以上
- **技能解锁**：解锁觉醒专属技能
- **形态变化**：获得血脉专属形态
- **属性大幅提升**：全属性大幅提升
- **特殊能力**：获得血脉专属特殊能力

### 6.3 觉醒失败

- **失败惩罚**：血脉纯度下降10%
- **冷却时间**：失败后需要等待10年（游戏内时间）才能再次尝试
- **材料消耗**：觉醒材料会消耗
- **成功率提升**：多次失败后成功率会逐渐提升

## 八、血脉传承

### 8.1 传承机制

**血脉池计算**：
- 父亲血脉比例 × 2
- 母亲血脉比例 × 2
- 两者相加得出孩子的血脉池

**血脉抽取**：
- 以12.5%为最小血脉单位
- 从血脉池中按照血脉占比权重抽取
- 每次抽取12.5%的一份血脉
- 重复抽取直到达到100%

**计算示例**：
```
父亲血脉：12.5%凤凰血脉，87.5%的人族血脉
母亲血脉：25%青龙血脉，75%的人族血脉

父亲血脉×2：25%凤凰血脉，175%的人族血脉
母亲血脉×2：50%青龙血脉，150%的人族血脉

血脉池合计：
- 凤凰血脉：25%
- 青龙血脉：50%
- 人族血脉：325%

按照权重抽取（每份12.5%）：
总份额 = 100% ÷ 12.5% = 8份

抽取结果（示例）：
- 人族血脉：50%（4份）
- 青龙血脉：37.5%（3份）
- 凤凰血脉：12.5%（1份）

孩子最终血脉构成：50%人族，37.5%青龙，12.5%凤凰
```

### 8.2 传承优化

- **血脉融合**：多种血脉融合可能产生新的血脉
- **血脉觉醒**：传承过程中有概率觉醒隐藏血脉
- **血脉强化**：父母血脉纯度越高，传承效果越好
- **特殊传承**：特定血脉组合有特殊传承效果

### 8.3 血脉变异

- **随机变异**：传承过程中有小概率发生血脉变异
- **变异效果**：获得全新的血脉能力
- **变异血脉**：可能产生从未见过的特殊血脉
- **变异条件**：父母血脉纯度高、血脉种类多的情况下更容易变异

- 克制方：对被克制方造成伤害+20%
- 被克制方：对克制方造成伤害-10%

### 12.3 稀有度平衡

- **普通血脉**：获取容易，成长稳定
- **稀有血脉**：获取困难，成长潜力大

- **成长曲线**：合理的血脉成长曲线

## 十四、总结

血脉系统是游戏的核心养成系统之一，通过丰富的血脉类型、纯度系统、技能系统、觉醒系统和进化系统，为玩家提供深度的养成体验。血脉系统与其他系统紧密结合，形成完整的角色成长体系，增强游戏的策略性和可玩性。

这个血脉系统设计为游戏提供了以下特色：

1. **多样性**：8种基础血脉，5种特殊血脉，多种变异血脉
2. **深度**：纯度、潜力、技能、觉醒、进化多层次养成
3. **策略性**：不同血脉搭配产生不同效果
4. **传承性**：通过繁育系统延续血脉，形成家族特色
5. **视觉表现**：血脉纯度和觉醒的视觉效果
6. **平衡性**：各血脉各有特色，相互克制

血脉系统将成为游戏的核心玩法之一，为玩家提供丰富的养成体验和策略选择。
