# Figma提示词 - 战斗系统

## 模块概述
战斗系统是游戏的核心玩法界面，包含战场视图、单位选择、技能释放、战斗结算等完整流程。

---

## 1. 战斗主界面（Battle Main Interface）

### 整体布局
```
┌─────────────────────────────────────────────────────────────┐
│ [回合数] [敌方剩余] [我方剩余]        [暂停] [自动] [加速] │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                    ┌───┐ ┌───┐ ┌───┐                      │
│                    │ E │ │   │ │ E │   E=敌方单位          │
│                    └───┘ └───┘ └───┘                      │
│                                                             │
│         ┌───┐           ┌───┐           ┌───┐            │
│         │   │           │ E │           │   │            │
│         └───┘           └───┘           └───┘            │
│                                                             │
│                    ┌───┐ ┌───┐ ┌───┐                      │
│                    │ P │ │ P │ │ P │   P=我方单位          │
│                    └───┘ └───┘ └───┘                      │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│ [角色1][角色2][角色3][角色4]  │  [攻击][技能][物品][移动]  │
│ HP ████████ MP ████████     │  [待机][撤退]               │
└─────────────────────────────────────────────────────────────┘
```

### 提示词

```
创建一个修仙题材回合制战棋游戏的战斗界面设计。

布局要求：
1. 顶部信息栏（高度48px，深色半透明背景）：
   - 左侧：当前回合数（圆形徽章），敌方剩余数量，我方剩余数量
   - 右侧：暂停按钮（三条横线），自动战斗按钮（播放图标），加速按钮（1x/2x）

2. 中央战场（占据主要空间）：
   - 15x15格子网格视图
   - 格子：半透明边框，可选择格子高亮
   - 地形：不同颜色/纹理表示不同地形
   - 单位：以Q版角色头像显示（48x48）
   - 我方单位：蓝色边框，敌方单位：红色边框
   - BOSS单位：有金色光晕特效

3. 底部操作区（高度120px）：
   - 左侧角色面板（宽度200px）：
     * 4个角色头像横向排列
     * 当前选中角色头像放大
     * HP条（红色渐变）+ MP条（蓝色渐变）
     * 角色名称和境界
   
   - 中央操作按钮（横向排列）：
     * 攻击（剑图标）
     * 技能（闪电图标）
     * 物品（背包图标）
     * 移动（脚印图标）
     * 待机（月亮图标）
     * 撤退（箭头图标）

4. 右侧单位详情（宽度180px，可折叠）：
   - 选中单位的详细属性
   - 技能列表（显示冷却）

设计风格：
- 战场背景：淡雅山水画风格
- 格子：淡金色半透明边框（rgba(255,215,0,0.3)）
- 可移动格子：淡蓝色高亮（rgba(100,149,237,0.4)）
- 可攻击格子：淡红色高亮（rgba(220,20,60,0.4)）
- 地形：不同颜色渐变表示

交互效果：
- 选中单位有呼吸光效
- 可移动范围有闪光点阵动画
- 攻击范围有红色圆圈扩散动画
- 伤害数字飘字上浮动画
- 角色死亡有消散动画
```

## 2. 技能选择面板（Skill Selection Panel）

### 提示词

```
创建一个修仙战棋游戏的技能选择面板（弹出式）。

布局要求：
1. 整体面板（宽度400px，高度auto，出现在屏幕中央）

2. 顶部：
   - 标题：选择技能
   - 当前MP显示：图标+数值

3. 技能列表（可滚动区域）：
   每行技能项（高度64px）：
   - 左侧：技能图标（48x48，圆形）
   - 中间：
     * 技能名称（粗体）
     * MP消耗（蓝色小字）
     * 技能描述（灰色小字，单行截断）
   - 右侧：
     * 冷却状态（如果冷却中显示剩余回合）
     * 选择按钮（箭头图标）

4. 底部：
   - 取消按钮（次要样式）

设计风格：
- 面板：白色圆角背景，阴影
- 技能图标：根据五行属性变色（金白、木绿、水蓝、火红、土黄）
- 冷却中技能：灰度显示，覆盖冷却遮罩
- MP不足：技能显示为暗色，图标变灰

交互效果：
- 面板从底部滑入动画
- 技能项hover有金色边框
- 点击技能项有选中高亮
- 冷却数字倒计时动画
- MP不足时点击有抖动提示
```

## 3. 目标选择界面（Target Selection Interface）

```
设计风格：

- 目标信息卡：白底金边，阴影

交互效果：

- 选中目标有金光扩散

```

## 4. 战斗结算界面（Battle Result Interface）

### 提示词

```
创建一个修仙战棋游戏的战斗结算界面。

布局要求：
1. 整体覆盖层（半透明黑色遮罩）

2. 中央结算卡片（宽度480px）：
   - 顶部装饰：云纹+胜利/失败标识
   - 胜利时：金色光芒 + 胜利大字
   - 失败时：灰色 + 失败大字

3. 战斗评价（3星评级系统）：
   - 3星：完美通关（金色星星）
   - 2星：普通通关（银色星星）
   - 1星：勉强通关（铜色星星）
   - 无星：失败（灰色空星）

4. 获得奖励列表：
   - 经验值：绿色数字 + 角色头像
   - 灵石：金色数字 + 宝石图标
   - 装备：紫色数字 + 装备图标
   - 材料：蓝色数字 + 材料图标

5. 角色升级提示（如有）：
   - 升级角色头像
   - 属性提升数字（绿色箭头+数值）

6. 底部按钮：
   - 再来一次（次要按钮）
   - 继续（主要按钮，金色）

设计风格：
- 胜利：金色光芒放射状背景
- 失败：灰色暗淡背景
- 星星：有从灰到金的点亮动画
- 奖励数字：有弹跳增加动画效果

交互效果：
- 面板从中央放大出现
- 奖励逐个显示（间隔200ms）
- 星星从左到右依次点亮
- 升级数字有跳动效果
- 按钮hover有光晕放大效果
```

## 6. 战斗信息显示（Battle Info Display）

### 提示词

```
创建一个修仙战棋游戏的战场信息显示组件。

布局要求：
1. 伤害数字（飘字）：
   - 普通伤害：白色数字
   - 暴击伤害：金色数字 + 感叹号
   - 治疗数字：绿色数字 + 加号
   - 护盾数字：蓝色数字 + 盾牌图标
   - 伤害减免：灰色数字 + 减号

2. 状态图标（单位头顶）：
   - 增益状态：绿色向上箭头
   - 减益状态：红色向下箭头
   - 控制状态：紫色漩涡图标
   - 持续伤害：红色火焰图标
   - 持续治疗：绿色笑脸图标

3. 行动顺序条（屏幕右侧）：
   - 垂直排列的圆形头像
   - 当前行动者高亮金色
   - 下一步行动者有轮廓光

4. 战斗日志（屏幕左侧，可折叠）：
   - 最近6条行动记录
   - 格式：攻击者 → 目标 + 伤害数值
   - 不同类型不同颜色标识

设计风格：
- 伤害数字：黑边白字描边
- 飘字动画：从目标位置向上飘动消失
- 状态图标：圆形小图标，24x24
- 行动顺序：圆形头像，32x32

交互效果：
- 伤害数字从目标位置向上飘出
- 暴击有屏幕震动效果
- 状态图标有缓慢旋转动画
- 行动顺序有条状流动动画
```

## 8. 阵法选择界面（Formation Selection Interface）

### 提示词

```
创建一个修仙战棋游戏的阵法选择界面。

布局要求：
1. 整体面板（宽度500px）

2. 阵法列表（横向排列4种阵法）：
   - 锋矢阵：攻击阵型，前排输出
   - 方圆阵：防御阵型，全面防护
   - 长蛇阵：机动阵型，灵活走位
   - 八卦阵：法防阵型，术法减免

3. 每种阵法显示：
   - 阵法图标（圆形64x64）
   - 阵法名称
   - 阵法效果描述
   - 位置示意图（小圆点表示站位）

4. 阵法详细信息：
   - 阵法效果详细属性加成
   - 适合的职业组合建议

5. 当前阵法效果预览：
   - 显示选择阵法后队伍的整体属性加成

设计风格：
- 阵法图标：根据阵法特性配色
- 未选中：灰度 + 虚线边框
- 选中：实线金框 + 光晕
- 位置示意图：圆点+连线，简洁明了

交互效果：
- 点击阵法有旋转选中效果
- 阵法效果有数值动画变化
- 位置示意图有闪烁动画
- 选中阵法有光芒扩散效果
```
