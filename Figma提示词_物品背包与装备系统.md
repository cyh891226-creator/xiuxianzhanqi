# Figma提示词 - 物品背包与装备系统

## 模块概述
物品背包系统是游戏的核心资源管理模块，包含背包管理、物品详情、装备穿戴、物品强化等功能。

---

### 提示词

```
创建一个修仙题材回合制战棋游戏的背包界面。

布局要求：
1. 顶部信息栏（高度56px）：
   - 左侧：返回按钮
   - 中央：标题"背包"
   - 右侧：整理按钮、筛选按钮

2. 容量信息（高度40px）：
   - 背包容量：当前数量/最大容量
   - 容量进度条
   - 灵石余额显示

3. 分类标签（高度48px）：
   - 全部 | 装备 | 道具 | 材料 | 凭证 | 时装
   - 当前选中标签高亮

4. 物品网格（可滚动区域）：
   - 5列网格布局
   - 每格物品卡片（64x64）：
     * 物品图标
     * 数量角标（右下角）
     * 品质边框色
   - 空格子：虚线框

5. 批量操作栏（底部）：
   - 一键出售按钮
   - 一键使用按钮
   - 批量选择按钮

设计风格：
- 背包格子：半透明深色背景
- 物品边框：根据品质变色（白/绿/蓝/紫/橙/红）
- 数量角标：圆形红色背景
- 选中物品：金色边框发光
- 空格子：虚线边框

交互效果：
- 点击物品有选中高亮
- 拖动物品有跟随效果
- 整理有物品归位动画
- 容量满有红色警告闪烁
```

## 2. 物品详情界面（Item Detail Interface）

### 提示词

```
创建一个修仙战棋游戏的物品详情弹窗。

布局要求：
1. 物品信息头部：
   - 大图标（96x96）
   - 物品名称（品质颜色）
   - 物品类型标签
   - 品质等级

2. 物品描述：
   - 详细文本说明
   - 效果数值

3. 物品属性（装备/道具）：
   - 属性列表
   - 数值显示
   - 稀有属性高亮

4. 获取途径：
   - 来源列表
   - 点击跳转

5. 操作按钮：
   - 使用（道具类）
   - 穿戴（装备类）
   - 出售（所有）
   - 锁定/解锁
   - 销毁

6. 锁定状态：
   - 锁定后有锁图标
   - 防止误操作

设计风格：
- 弹窗：白色圆角卡片
- 物品图标：品质边框
- 锁定：右上角锁图标
- 属性：金色数字
- 按钮：按功能分颜色

交互效果：
- 弹窗从中央放大出现
- 点击操作有确认反馈
- 锁定有锁动画
- 获得途径可点击跳转
```
