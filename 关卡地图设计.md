# 修仙战棋 - 关卡地图设计

## 一、地图系统概述

### 1.1 设计理念
关卡地图是修仙战棋的舞台，融合修仙世界观、策略战斗、探索发现，打造沉浸式体验。

### 1.2 地图类型
- **主线关卡**：推进剧情，固定设计
- **支线关卡**：丰富内容，可选挑战
- **秘境探索**：随机生成，roguelike元素
- **活动关卡**：限时开放，特殊奖励

## 二、地形系统

### 2.1 基础地形

| 地形 | 移动消耗 | 效果 | 视觉表现 |
|------|----------|------|----------|
| 平地 | 1 | 无 | 草地/石板 |
| 山地 | 2 | 防御+20%，移动力-1 | 岩石/山峰 |
| 森林 | 2 | 闪避+15%，视野受限 | 树木/灌木 |
| 水域 | 3 | 闪避-10%，部分单位无法通过 | 河流/湖泊 |
| 沼泽 | 3 | 每回合损失5%生命 | 泥潭/湿地 |

### 2.2 特殊地形

| 地形 | 效果 | 适用场景 |
|------|------|----------|
| 灵脉 | 每回合恢复5%法力 | 修炼圣地 |
| 毒瘴 | 每回合可能中毒 | 毒修领地 |
| 火域 | 火属性伤害+30% | 火山/熔岩 |
| 冰原 | 水属性技能冰冻概率+20% | 极寒之地 |
| 雷云 | 每回合随机雷击 | 天劫场景 |
| 迷雾 | 视野大幅降低 | 迷阵/幻境 |
| 传送阵 | 可传送至其他传送阵 | 秘境/洞府 |
| 陷阱 | 触发后造成伤害或异常 | 古墓/禁地 |

### 2.3 地形互动

#### 可破坏地形
- **山体**：可被强力攻击破坏，造成落石伤害
- **树木**：可被火系技能点燃，造成持续伤害
- **冰面**：可被火系技能融化，变为水域
- **桥梁**：可被攻击破坏，阻断路径

#### 动态地形
- **潮汐**：水域范围周期性变化
- **岩浆**：火域范围缓慢扩散
- **毒雾**：毒瘴范围随风向移动
- **雷暴**：雷云位置随机移动

## 三、场景类型

### 3.1 门派场景

#### 昆仑仙宗
- **风格**：仙气飘渺，剑气纵横
- **地形**：山地为主，灵脉分布
- **特色**：剑阵机关，御剑飞行点
- **敌人**：剑修弟子、护山灵兽

#### 太虚观
- **风格**：丹香弥漫，云雾缭绕
- **地形**：平地为主，药田遍布
- **特色**：丹炉机关，恢复灵泉
- **敌人**：丹修弟子、药人

#### 天机阁
- **风格**：星象密布，机关重重
- **地形**：复杂迷宫，传送阵多
- **特色**：阵法机关，陷阱密布
- **敌人**：阵修弟子、机关兽

#### 万佛寺
- **风格**：佛光普照，庄严肃穆
- **地形**：开阔广场，佛像林立
- **特色**：佛光加持，净化区域
- **敌人**：体修弟子、护法金刚

### 3.2 野外场景

#### 东荒妖林
- **风格**：蛮荒原始，妖气冲天
- **地形**：森林、沼泽混合
- **特色**：妖兽巢穴，毒瘴区域
- **敌人**：各种妖兽、妖修

#### 西漠荒原
- **风格**：黄沙漫天，荒凉孤寂
- **地形**：沙地、绿洲交错
- **特色**：沙尘暴，海市蜃楼
- **敌人**：沙妖、盗匪

#### 南蛮毒沼
- **风格**：毒瘴弥漫，危机四伏
- **地形**：沼泽、毒林为主
- **特色**：剧毒区域，蛊虫巢穴
- **敌人**：蛊修、毒虫

#### 北原冰原
- **风格**：冰雪世界，极寒刺骨
- **地形**：冰原、雪山为主
- **特色**：冰冻区域，雪崩
- **敌人**：冰妖、雪怪

### 3.3 秘境场景

#### 上古遗迹
- **风格**：古老神秘，机关重重
- **地形**：复杂迷宫，陷阱密布
- **特色**：古代机关，守护傀儡
- **奖励**：上古功法、稀有材料

#### 洞天福地
- **风格**：灵气浓郁，仙家宝地
- **地形**：灵脉汇聚，灵泉遍布
- **特色**：修炼加速，灵药生长
- **奖励**：灵石、丹药、功法

#### 魔道禁地
- **风格**：阴森恐怖，魔气冲天
- **地形**：血池、骨海、魔焰
- **特色**：魔化效果，心智考验
- **奖励**：魔道功法、邪器

#### 天劫之地
- **风格**：雷霆万钧，天威浩荡
- **地形**：雷云密布，天雷滚滚
- **特色**：天雷轰击，渡劫考验
- **奖励**：渡劫成功，境界突破

## 四、关卡设计

### 4.1 主线关卡示例

#### 第一章：初入修仙

**关卡1-1：拜师之路**
- **目标**：到达山门
- **地图**：10×10，山路地形
- **敌人**：无/野兽
- **教学**：移动、攻击基础

**关卡1-2：入门试炼**
- **目标**：击败试炼傀儡
- **地图**：12×12，演武场
- **敌人**：试炼傀儡×3
- **教学**：技能使用

**关卡1-3：同门切磋**
- **目标**：击败同门师兄
- **地图**：12×12，演武场
- **敌人**：同门师兄×1
- **教学**：策略战斗

#### 第二章：下山历练

**关卡2-1：妖兽袭击**
- **目标**：保护村庄
- **地图**：15×15，村庄地形
- **敌人**：妖狼群
- **机制**：保护NPC

**关卡2-2：古墓探险**
- **目标**：寻找宝物
- **地图**：15×15，古墓迷宫
- **敌人**：僵尸、机关
- **机制**：解谜、陷阱

**关卡2-3：魔道初现**
- **目标**：击退魔修
- **地图**：15×15，荒野
- **敌人**：魔道弟子
- **机制**：首次对战魔道

### 4.2 BOSS关卡设计

#### 金丹期BOSS：赤焰妖虎

**场景**：火山洞穴
**地图**：15×15，火域地形

**BOSS机制**：
- **烈焰吐息**：直线范围火焰伤害
- **狂暴**：生命值低于50%时攻击翻倍
- **召唤**：召唤小火妖协助战斗
- **地形互动**：点燃周围地形

**攻略要点**：
- 利用水域地形降低火伤
- 优先清理小火妖
- 控制BOSS狂暴时机

#### 元婴期BOSS：幽冥鬼王

**场景**：幽冥深渊
**地图**：18×18，鬼气地形

**BOSS机制**：
- **万鬼噬魂**：召唤鬼魂攻击
- **恐惧咆哮**：范围恐惧效果
- **灵魂收割**：击杀单位恢复生命
- **不死**：首次死亡后复活

**攻略要点**：
- 携带净化技能
- 避免单位被击杀
- 准备二次战斗

## 五、随机事件系统

### 5.1 探索事件

| 事件类型 | 触发条件 | 效果 |
|----------|----------|------|
| 发现灵泉 | 移动到特定格子 | 恢复全部生命和法力 |
| 遭遇妖兽 | 随机触发 | 进入战斗 |
| 发现宝箱 | 探索隐藏区域 | 获得随机奖励 |
| 遇到散修 | 随机触发 | 可交易、招募或战斗 |
| 触发陷阱 | 移动到陷阱格 | 受到伤害或异常 |
| 发现秘境 | 特定条件触发 | 进入特殊关卡 |

### 5.2 随机事件示例

#### 神秘商人
```
你在路上遇到一位神秘商人，他提供以下交易：
1. 用100灵石购买随机丹药
2. 用稀有材料交换功法秘籍
3. 拒绝交易并离开
```

#### 求救信号
```
你发现远处有求救信号，似乎有修士遇险：
1. 前往救援（可能获得奖励或陷入陷阱）
2. 置之不理（无影响）
3. 暗中观察（发现是魔道陷阱）
```

#### 上古遗迹
```
你发现一处上古遗迹入口：
1. 进入探索（进入秘境关卡）
2. 标记位置，以后再来
3. 离开（担心危险）
```

## 六、关卡编辑器

### 6.1 编辑器功能

#### 基础功能
- 地图尺寸设置（10×10 到 20×20）
- 地形绘制（画笔、填充、橡皮擦）
- 单位放置（玩家、敌人、NPC）
- 机关设置（门、开关、传送阵）

#### 高级功能
- 事件脚本编辑
- 触发器设置
- 关卡流程设计
- 测试运行

### 6.2 关卡数据结构

```csharp
[System.Serializable]
public class LevelData
{
    public string levelId;
    public string levelName;
    public string description;
    
    [Header("地图")]
    public int width = 15;
    public int height = 15;
    public CellData[,] cells;
    
    [Header("单位")]
    public List<UnitPlacement> playerUnits;
    public List<UnitPlacement> enemyUnits;
    public List<UnitPlacement> npcUnits;
    
    [Header("目标")]
    public VictoryCondition victoryCondition;
    public List<Objective> objectives;
    
    [Header("事件")]
    public List<LevelEvent> events;
    
    [Header("奖励")]
    public List<Reward> rewards;
}

[System.Serializable]
public class CellData
{
    public Vector2Int position;
    public CellType cellType;
    public bool isWalkable = true;
    public string specialEffect;
}

[System.Serializable]
public class UnitPlacement
{
    public string unitId;
    public Vector2Int position;
    public Faction faction;
    public int level = 1;
}
```

## 七、关卡平衡

### 7.1 难度曲线

| 章节 | 推荐境界 | 敌人强度 | 关卡复杂度 |
|------|----------|----------|------------|
| 第一章 | 练气期 | ★☆☆☆☆ | 简单 |
| 第二章 | 筑基期 | ★★☆☆☆ | 简单 |
| 第三章 | 金丹期 | ★★★☆☆ | 中等 |
| 第四章 | 元婴期 | ★★★★☆ | 中等 |
| 第五章 | 化神期 | ★★★★★ | 困难 |

### 7.2 关卡节奏

#### 战斗节奏
- **短关卡**：5-10分钟，1-2场战斗
- **中关卡**：15-20分钟，3-4场战斗
- **长关卡**：30分钟以上，BOSS战

#### 休息点设计
- 每2-3场战斗设置恢复点
- 提供存档、治疗、整备功能
- 剧情推进节点

## 八、总结

关卡地图系统为修仙战棋提供了丰富的战斗场景：

1. **地形策略**：不同地形影响战斗策略
2. **场景多样**：门派、野外、秘境多种场景
3. **关卡设计**：主线、支线、BOSS多种类型
4. **随机事件**：增加探索乐趣和重玩价值
5. **编辑器支持**：便于快速创建和调整关卡

这个系统为游戏提供了丰富的内容和长期的可玩性。
