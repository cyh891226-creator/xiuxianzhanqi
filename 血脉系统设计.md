# 天地劫—万世仙宗 - 血脉系统设计

## 一、血脉系统概述

### 1.1 设计理念

血脉系统是游戏的核心养成系统之一，通过血脉的传承，为角色提供独特的属性加成和特性，增强游戏的策略性和可玩性。血脉系统与角色属性系统、繁育系统紧密结合，形成完整的角色成长体系。

### 1.2 核心机制

- **血脉类型**：不同种族的血脉，根据地域分布设计
- **血脉纯度**：0-100%，影响血脉潜力效果强度
- **血脉潜力**：影响六维属性的成长潜力
- **血脉特性**：血脉专属特性，稀有血脉和传说血脉拥有
- **血脉传承**：通过繁育系统传承血脉给后代

### 3.1 纯度等级

| 纯度范围 | 等级 | 效果 | 视觉表现 |

| 41-60% | 中等 | 明显效果 | 明显光晕 |

| 81-100% | 纯粹 | 极致效果 | 血脉特效显现 |

### 3.2 纯度影响

- **属性加成**：纯度越高，血脉潜力对属性成长的影响越强
- **特性效果**：纯度越高，血脉特性效果越强
- **传承效果**：纯度越高，传承给后代的血脉比例越高

**潜力计算公式**：
```
血脉潜力 = 基础潜力 × 纯度修正
```

**基础潜力**：每种血脉对六维属性的固有潜力值

**纯度修正**：
- 微弱：0.5
- 稀薄：0.7

## 六、血脉传承

### 6.1 传承机制

**血脉池计算**：
- 父亲血脉比例 × 2
- 母亲血脉比例 × 2
- 两者相加得出孩子的血脉池

**血脉抽取**：
- 以12.5%为最小血脉单位
- 从血脉池中按照血脉占比权重抽取
- 每次抽取12.5%的一份血脉
- 重复抽取直到达到100%

**计算示例**：
```
父亲血脉：12.5%凤凰血脉，87.5%的人族血脉
母亲血脉：25%龙族血脉，75%的人族血脉

父亲血脉×2：25%凤凰血脉，175%的人族血脉
母亲血脉×2：50%龙族血脉，150%的人族血脉

血脉池合计：
- 凤凰血脉：25%
- 龙族血脉：50%
- 人族血脉：325%

按照权重抽取（每份12.5%）：
总份额 = 100% ÷ 12.5% = 8份

抽取结果（示例）：
- 人族血脉：50%（4份）
- 龙族血脉：37.5%（3份）
- 凤凰血脉：12.5%（1份）

孩子最终血脉构成：50%人族，37.5%龙族，12.5%凤凰
```

### 6.2 传承优化

- **血脉融合**：多种血脉融合可能产生新的血脉
- **纯度遗传**：父母血脉纯度越高，传承效果越好
- **特殊传承**：特定血脉组合有特殊传承效果

## 七、血脉与其他系统的关联

### 7.1 与属性系统的关联

- **潜力计算**：血脉潜力影响属性潜力

- **成长影响**：血脉影响属性成长速率

- **血脉传承**：通过繁育传承血脉
- **纯度遗传**：父母血脉纯度影响后代血脉纯度
- **相亲成功率**：部分血脉特性提升相亲成功率

### 7.3 与装备系统的关联

- **血脉专属装备**：特定血脉可使用专属装备
- **装备属性**：血脉影响装备属性效果
- **法宝共鸣**：血脉与特定法宝产生共鸣

### 7.4 与世界地图的关联

- **血脉任务**：特定血脉可接取专属任务
- **血脉圣地**：特定血脉有专属的修炼圣地

## 十一、总结

血脉系统是游戏的核心养成系统之一，通过丰富的血脉类型、纯度系统、特性系统和传承机制，为玩家提供深度的养成体验。血脉系统与其他系统紧密结合，形成完整的角色成长体系，增强游戏的策略性和可玩性。

这个血脉系统设计为游戏提供了以下特色：

1. **多样性**：根据地域分布设计多种血脉类型
2. **深度**：纯度、潜力、特性多层次养成
3. **策略性**：不同血脉搭配产生不同效果
4. **传承性**：通过繁育系统延续血脉，形成家族特色
5. **视觉表现**：血脉纯度的视觉效果
6. **平衡性**：各血脉各有特色，相互补充

血脉系统将成为游戏的核心玩法之一，为玩家提供丰富的养成体验和策略选择。
