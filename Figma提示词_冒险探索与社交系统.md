# Figma提示词 - 冒险探索与社交系统

## 模块概述
冒险探索系统包含大地图、关卡选择、秘境探索等功能。社交系统包含好友、公会、聊天等功能。

---

### 提示词

```
创建一个修仙题材回合制战棋游戏的大地图界面。

布局要求：
1. 顶部导航栏：
   - 左侧：返回按钮
   - 中央：标题"大陆地图"
   - 右侧：缩放按钮、筛选按钮

2. 大地图区域（主要空间）：
   - 2D俯视风格地图
   - 各区域用不同颜色/纹理区分
   - 章节节点用图标标记
   - 已通关节点：实心
   - 当前节点：金色脉冲
   - 未解锁节点：灰色锁定
   - BOSS节点：特殊图标

3. 章节信息栏（底部）：
   - 当前章节名称
   - 关卡进度显示
   - 章节难度星级

4. 功能入口（底部图标栏）：
   - 关卡：进入章节关卡
   - 秘境：进入秘境探索
   - 活动：限时活动入口
   - 奖励：奖励中心
   - 剧情：剧情回顾

5. 地图缩放与拖拽：
   - 支持双指缩放
   - 支持拖拽查看

设计风格：
- 地图：淡雅水墨山水风格
- 已通关：蓝色节点
- 当前：金色脉冲光晕
- 未解锁：灰色虚线
- BOSS：红色特殊标记
- 连接线：虚线路径

交互效果：
- 点击节点有放大详情
- 解锁新节点有金光出现
- 当前节点有呼吸动画
- 拖拽有跟随流畅效果
- 缩放有平滑过渡动画
```

## 2. 关卡选择界面（Stage Selection Interface）

### 提示词

```
创建一个修仙战棋游戏的关卡选择界面。

布局要求：
1. 章节信息头部：
   - 章节名称
   - 章节描述
   - 星级评价（已获得）
   - 章节进度

2. 关卡列表（垂直滚动）：
   每关卡节点：
   - 关卡编号
   - 关卡名称
   - 难度星级（1-5）
   - 通关状态
   - 体力消耗
   - 首通奖励图标

3. 关卡详情（点击后）：
   - 关卡名称
   - 敌人信息预览
   - 地形预览
   - 推荐战力
   - 通关奖励详情
   - 扫荡按钮（如已通关）

4. BOSS关卡（特殊样式）：
   - 大尺寸节点
   - BOSS立绘预览
   - BOSS技能说明
   - 首通奖励预览

5. 快速操作：
   - 扫荡全部（已通关）
   - 体力不足提示

设计风格：
- 普通关卡：圆形节点
- BOSS关卡：大号特殊节点
- 已通关：绿色勾选
- 当前可打：蓝色高亮
- 未解锁：灰色锁定
- 难度星级：金色星星

交互效果：
- 点击节点展开详情
- 通关有星光动画
- 首通奖励有掉落动画
- 扫荡有快速动画展示
- 解锁新关卡有出现动画
```

## 4. 游戏设置界面（Game Settings Interface）

### 提示词

```
创建一个修仙战棋游戏的游戏设置界面（单机版）。

布局要求：
1. 顶部标签栏：
   - 画面 | 声音 | 控制 | 存档 | 语言

2. 画面设置：
   - 分辨率选择
   - 画质：低/中/高/极致
   - 帧率：30/60/无限制
   - 全屏模式：开关
   - 垂直同步：开关
   - 抗锯齿：开关

3. 声音设置：
   - 音乐音量：0-100滑动条
   - 音效音量：0-100滑动条
   - 语音音量：0-100滑动条
   - 主音量：0-100滑动条

4. 控制设置：
   - 键盘映射（可自定义）
   - 鼠标灵敏度
   - 快捷键设置
   - 控制器支持：开关

5. 存档设置：
   - 自动存档：开关
   - 存档位置：显示存档文件夹路径
   - 备份存档：按钮
   - 恢复存档：按钮

6. 语言设置：
   - 语言选择：中文/英文
   - 文字大小：小/中/大

7. 底部按钮：
   - 保存设置
   - 恢复默认
   - 关闭

设计风格：
- 标签页：下划线样式
- 滑动条：古风样式轨道
- 开关：圆形toggle
- 按钮：圆角矩形

交互效果：
- 切换设置有即时预览
- 保存有确认提示
- 恢复默认有二次确认
- 存档操作有进度提示
```
