# 修仙战棋 - 技能系统设计

## 一、技能体系概述

### 1.1 设计理念
技能系统是修仙战棋的核心战斗元素，融合五行相生相克、职业特色、境界层次，打造深度策略体验。

### 1.2 技能分类
- **普通技能**：基础战斗技能，消耗法力
- **绝技**：强力技能，消耗怒气
- **被动技能**：自动生效，无需激活
- **天赋技能**：职业专属，独特机制
- **合击技**：多人配合，威力倍增


**剑修核心技能详解：**

#### 御剑术系
- **御剑术（入门）**：远程物理攻击，距离3格
- **飞剑斩（进阶）**：御剑术升级版，距离5格，可穿透1个目标
- **万剑阵（精通）**：范围攻击，对区域内所有敌人造成伤害

#### 剑气系
- **剑气（入门）**：普通攻击附加剑气伤害
- **剑芒（进阶）**：攻击范围+1，伤害+20%
- **剑意（精通）**：暴击时额外造成50%伤害

#### 剑体系
- **剑体（入门）**：防御+10%，生命+10%
- **金刚（进阶）**：受到致命伤害时保留1点生命，冷却5回合
- **不坏（精通）**：免疫控制效果

#### 终极技能
- **剑心通明（天赋）**：攻击无视目标50%防御
- **万剑归宗（绝技）**：全屏攻击，每把剑造成独立伤害

## 五、技能平衡设计

### 5.1 伤害公式
```
技能伤害 = (攻击者属性 × 技能系数 - 防御者属性) × 克制系数 × 随机系数

随机系数：0.9 - 1.1
```

### 5.2 消耗设计
- **低阶技能**：消耗10-20法力，冷却1-2回合
- **中阶技能**：消耗30-50法力，冷却3-4回合
- **高阶技能**：消耗60-100法力，冷却5-8回合
- **绝技**：消耗满怒气，冷却10回合以上

### 5.3 平衡原则
1. **高风险高回报**：消耗大的技能伤害高
2. **职业互补**：没有全能职业，需要团队配合
3. **克制关系**：五行相克影响技能效果
4. **成长曲线**：技能随境界提升而增强

## 六、技能特效与表现

### 6.1 特效设计原则
- 符合五行属性特征
- 体现职业特色
- 不遮挡战场信息
- 性能优化考虑

### 6.2 特效示例
- **金系**：剑光、金属碎片、金色光芒
- **木系**：绿叶、藤蔓、生命光芒
- **水系**：水流、冰晶、蓝色波纹
- **火系**：火焰、爆炸、红色光芒
- **土系**：岩石、尘土、棕色光芒

## 七、技能获取与升级

### 7.1 技能获取方式
- **门派学习**：加入门派学习基础技能
- **秘籍获取**：探索获得稀有技能
- **境界突破**：突破境界解锁新技能
- **任务奖励**：完成特定任务获得

### 7.2 技能升级
- 使用技能获得熟练度
- 熟练度满后消耗资源升级
- 升级提升效果、降低消耗
- 最高10级

## 八、总结

技能系统是修仙战棋的核心战斗机制：

1. **五行体系**：金木水火土相生相克，策略深度
2. **职业特色**：七大流派，各有独特技能树
3. **合击技**：团队配合，创造强力组合
4. **平衡设计**：确保战斗公平性和策略性
5. **成长系统**：技能升级，长期养成目标

这个技能系统为游戏提供了丰富的战斗策略和深度。
