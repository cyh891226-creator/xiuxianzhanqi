# 天地劫—万世仙宗 - 血脉系统设计

## 一、血脉系统概述

### 1.1 设计理念

血脉系统是游戏的核心养成系统之一，通过血脉的传承，为角色提供独特的属性加成和特性，增强游戏的策略性和可玩性。血脉系统与角色属性系统、繁育系统紧密结合，形成完整的角色成长体系。

### 1.2 核心机制

- **血脉类型**：不同种族的血脉，根据地域分布设计
- **血脉纯度**：0-100%，影响血脉潜力效果强度
- **血脉潜力**：影响六维属性的成长潜力
- **血脉特性**：血脉专属特性，稀有血脉和传说血脉拥有
- **血脉传承**：通过繁育系统传承血脉给后代

- **传承效果**：纯度越高，传承给后代的血脉比例越高

## 四、血脉潜力

### 4.1 潜力计算

**潜力计算公式**：
```
血脉潜力 = 基础潜力 × 纯度修正
```

**基础潜力**：每种血脉对六维属性的固有潜力值

**纯度修正**：
- 微弱：0.5
- 稀薄：0.7
- 中等：0.9

## 六、血脉传承

### 6.1 传承机制

**血脉池计算**：
- 父亲血脉比例 × 2
- 母亲血脉比例 × 2
- 两者相加得出孩子的血脉池

**血脉抽取**：
- 以12.5%为最小血脉单位
- 从血脉池中按照血脉占比权重抽取
- 每次抽取12.5%的一份血脉
- 重复抽取直到达到100%

**计算示例**：
```
父亲血脉：12.5%凤凰血脉，87.5%的人族血脉
母亲血脉：25%龙族血脉，75%的人族血脉

父亲血脉×2：25%凤凰血脉，175%的人族血脉
母亲血脉×2：50%龙族血脉，150%的人族血脉

血脉池合计：
- 凤凰血脉：25%
- 龙族血脉：50%
- 人族血脉：325%

按照权重抽取（每份12.5%）：
总份额 = 100% ÷ 12.5% = 8份

抽取结果（示例）：
- 人族血脉：50%（4份）
- 龙族血脉：37.5%（3份）
- 凤凰血脉：12.5%（1份）

孩子最终血脉构成：50%人族，37.5%龙族，12.5%凤凰
```

### 6.2 传承优化

- **血脉融合**：多种血脉融合可能产生新的血脉
- **纯度遗传**：父母血脉纯度越高，传承效果越好
- **特殊传承**：特定血脉组合有特殊传承效果

- **相亲成功率**：部分血脉特性提升相亲成功率

### 7.3 与装备系统的关联

- **血脉专属装备**：特定血脉可使用专属装备

### 7.4 与世界地图的关联

- **血脉任务**：特定血脉可接取专属任务
- **血脉圣地**：特定血脉有专属的修炼圣地

血脉系统将成为游戏的核心玩法之一，为玩家提供丰富的养成体验和策略选择。
