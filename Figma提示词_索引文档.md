### 字体规范
- 标题：思源宋体/楷体风格
- 正文：思源黑体
- 数字：清晰易读

### 尺寸规范
- 图标：24x24、48x48、64x64
- 头像：40x40、64x64、80x80、120x120
- 按钮高度：48px、56px、64px
- 卡片圆角：12px、16px

## 🚀 使用方法

1. **选择对应系统的文档**
2. **复制相应的提示词**
3. **在Figma中使用中文描述**
4. **根据游戏风格调整细节**
5. **添加交互效果说明**
6. **导出为设计稿**

---

## 📝 提示词使用示例

```
创建一个修仙题材回合制战棋游戏的[界面名称]。

布局要求：
1. [具体布局描述]
2. [元素位置和尺寸]
3. [交互元素列表]

设计风格：
- [颜色、字体、图标风格]
- [动效要求]

交互效果：
- [具体交互描述]
```

---

## ⚠️ 注意事项

1. **尺寸单位**：使用px作为单位
2. **颜色格式**：使用十六进制颜色码
3. **交互说明**：详细描述每个可交互元素
4. **动效描述**：说明动画效果类型
5. **平台适配**：考虑PC和移动端的不同
