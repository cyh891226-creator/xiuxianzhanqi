# Figma提示词 - 角色与伙伴系统

## 模块概述
角色系统是游戏的核心养成模块，包含角色详情、伙伴管理、技能学习、装备穿戴等功能。

---

### 提示词

```
创建一个修仙题材回合制战棋游戏的角色详情界面。

布局要求：
1. 顶部导航栏（高度56px）：
   - 左侧：返回箭头按钮
   - 中央：标题"角色详情"
   - 右侧：更多操作按钮（三点菜单）

2. 角色形象区（顶部区域，高度200px）：
   - 左侧：角色立绘/头像（大尺寸，160x200）
   - 右侧信息：
     * 角色名称（粗体大字）
     * 境界徽章 + 境界名称
     * 职业图标 + 职业名称
     * 所属门派

3. 属性面板（白色卡片）：
   - 两列布局显示属性
   - 左侧：生命、法力、攻击、防御
   - 右侧：速度、暴击、闪避、命中
   - 每项属性：图标 + 属性名 + 数值
   - 重要数值用金色高亮

4. 灵根血脉区（特殊样式卡片）：
   - 灵根属性条：五行颜色渐变
   - 血脉名称 + 品质标识
   - 血脉效果简短描述

5. 技能列表（横向滑动）：
   - 6个技能图标（56x56）
   - 普通技能、绝技、被动技能分类
   - 点击查看详情

6. 装备展示（横向排列）：
   - 武器、防具、饰品三个位置
   - 显示当前装备图标和名称
   - 空装备显示虚线框

7. 亲密度面板（进度条样式）：
   - 亲密度进度条 + 百分比
   - 亲密度等级显示
   - 羁绊点数

设计风格：
- 界面：暗色背景（#1a1a2e）
- 卡片：圆角矩形，半透明白色背景
- 境界徽章：根据境界高低变色
- 属性图标：五行配色（金白、木绿、水蓝、火红、土黄）
- 血脉：发光边框效果

交互效果：
- 角色立绘有轻微呼吸动画
- 属性数值有悬浮提示详情
- 技能图标hover有放大效果
- 装备栏点击可更换装备
- 亲密度提升有爱心动画
```

## 2. 伙伴列表界面（Partner List Interface）

### 提示词

```
创建一个修仙战棋游戏的伙伴列表界面。

布局要求：
1. 顶部筛选栏（高度48px）：
   - 全部 | 战斗 | 辅助 | 控制 | 治疗（横向标签）
   - 排序：等级 | 战力 | 亲密度（下拉选择）

2. 伙伴卡片网格（2列布局）：
   每个伙伴卡片（宽度50%，高度140px）：
   - 左侧：角色头像（80x80，圆形）
   - 右侧信息：
     * 名称 + 境界图标
     * 职业标签（彩色标签）
     * 战力数值
     * HP条 + MP条
     * 亲密度进度（简化条）

3. 空槽位显示：
   - 虚线框 + 加号
   - 点击可招募新伙伴

4. 快速操作：
   - 批量强化按钮
   - 一键装备按钮

设计风格：
- 卡片：白色圆角背景，阴影
- 头像框：根据境界颜色边框
- 职业标签：五行颜色背景
- 亲密度：爱心图标进度条

交互效果：
- 卡片hover有上浮阴影效果
- 点击有选中边框动画
- 新伙伴加入有闪光入场动画
- 滑动切换筛选有平滑过渡
```

## 3. 伙伴详情/编辑界面（Partner Edit Interface）

### 提示词

```
创建一个修仙战棋游戏的伙伴详情编辑界面。

布局要求：
1. 顶部标签页：
   - 详情 | 技能 | 装备 | 传承 | 故事

2. 详情标签页：
   - 与角色详情类似

3. 技能标签页：
   - 已学会技能列表
   - 可学习技能列表（灰色锁定）
   - 技能升级材料显示
   - 技能升级按钮

4. 装备标签页：
   - 当前装备详情
   - 可更换装备提示
   - 强化材料显示

5. 传承标签页：
   - 可传承的技能/属性
   - 传承对象选择
   - 传承确认按钮

6. 故事标签页：
   - 角色背景故事
   - 角色语音/台词
   - 角色立绘画廊

设计风格：
- 标签页：下划线样式，当前标签金色
- 技能项：卡片式，显示冷却和消耗
- 锁定技能：灰度 + 锁图标
- 故事文本：古风卷轴样式背景

交互效果：
- 标签切换有滑块动画
- 技能升级有光芒升级效果
- 传承有连线动画示意
- 立绘可点击放大查看
```

## 4. 伙伴招募界面（Partner Recruit Interface）

### 提示词

```
创建一个修仙战棋游戏的伙伴招募界面。

布局要求：
1. 招募类型选择（顶部）：

5. 保底机制说明：
   - 保底计数器

- 招募按钮：发光脉冲效果
- 普通招募：蓝色调

- 限定招募：金色脉冲

交互效果：
- 招募按钮点击有涟漪扩散
- 招募过程有召唤动画
- 获得高稀有度有全屏闪光
- 保底触发有特殊提示
```

## 5. 亲密度系统界面（Affinity System Interface）

### 提示词

```
创建一个修仙战棋游戏的亲密度系统界面。

布局要求：
1. 伙伴头像（居中，120x120）：
   - 大头像展示
   - 当前亲密度等级
   - 亲密度经验条

2. 亲密度等级奖励（列表）：
   - 等级1-10的奖励预览
   - 已解锁奖励（绿色勾选）
   - 当前可解锁（黄色高亮）
   - 未解锁（灰色锁）

3. 提升方式（横向卡片）：
   - 组队出战（+10亲密度）
   - 赠送礼物（+5/20/50）
   - 对话互动（+2亲密度）
   - 完成心愿（+30亲密度）

4. 羁绊技能（解锁后显示）：
   - 解锁的羁绊技能图标
   - 技能效果描述
   - 合击技解锁提示

5. 羁绊故事（解锁后）：
   - 解锁的羁绊剧情入口
   - 剧情进度显示

设计风格：
- 亲密度条：粉色渐变（爱心色）
- 已解锁：金色边框
- 当前可解锁：黄色脉冲
- 未解锁：灰色虚线框

交互效果：
- 亲密度提升有爱心飘出动画
- 升级有光芒爆发效果
- 解锁技能有技能图标飞入动画
- 羁绊故事有翻页效果
```

## 6. 角色升级界面（Character Level Up Interface）

### 提示词

```
创建一个修仙战棋游戏的角色升级界面。

布局要求：
1. 当前状态显示：
   - 角色头像
   - 当前等级 + 境界
   - 当前经验值 / 升级经验

2. 经验提升预览：
   - 获得经验条动画
   - 升级后可达到的等级预览

3. 升级材料选择：
   - 经验丹药列表
   - 每个丹药：图标 + 经验值 + 数量
   - 选择数量滑动条

4. 确认升级区：
   - 消耗显示
   - 升级按钮

5. 批量升级（可选）：
   - 使用全部丹药按钮
   - 预计升级等级显示

设计风格：
- 经验条：绿色渐变填充动画
- 丹药图标：瓶装样式，五行颜色
- 升级预览：金色数字变化
- 按钮：绿色渐变（表示成长）

交互效果：
- 添加丹药有数字跳动
- 经验条有流动填充动画
- 升级成功有金光爆发
- 等级数字有滚动效果
```

## 7. 角色突破界面（Character Breakthrough Interface）

### 提示词

```
创建一个修仙战棋游戏的角色境界突破界面。

布局要求：
1. 突破前状态：
   - 角色头像
   - 当前境界 + 小境界
   - 当前属性预览

2. 突破后预览：
   - 突破后境界显示
   - 属性提升幅度（箭头+数值）
   - 新技能解锁预览

3. 突破材料（列表）：
   - 所需材料图标 + 名称 + 数量
   - 已拥有数量 / 需求数量
   - 缺少材料提示（红色）

4. 突破成功率：
   - 成功率百分比（大字显示）
   - 使用道具可提升成功率
   - 突破丹选择

5. 突破按钮：
   - 条件满足：金色发光按钮
   - 条件不足：灰色禁用按钮

6. 风险提示（如有）：
   - 突破失败惩罚说明
   - 保护道具提示

设计风格：
- 突破前：蓝色调
- 突破后：金色光芒
- 材料充足：绿色勾选
- 材料不足：红色叉号
- 成功率：进度环样式

交互效果：
- 点击突破有天劫雷光动画
- 成功有金光冲天效果
- 失败有碎石散落效果
- 属性提升有数字滚动
```

## 8. 传承系统界面（Inheritance System Interface）

### 提示词

```
创建一个修仙战棋游戏的角色传承界面。

布局要求：
1. 传承说明（顶部）：
   - 传承类型选择：
     * 技能传承
     * 属性传承
     * 装备传承

2. 传承者选择（左侧）：
   - 可选角色列表
   - 角色头像 + 名称 + 等级
   - 传承内容预览

3. 传承内容设置（中央）：
   - 可传承项目勾选
   - 传承比例滑动条（10%-100%）
   - 传承消耗显示

4. 被传承者选择（右侧）：
   - 可选角色列表（通常是弟子）
   - 选择后显示继承效果预览

5. 传承预览：
   - 传承前/后对比
   - 属性变化数值

6. 确认传承按钮：
   - 二次确认弹窗

设计风格：
- 传承者：蓝色边框
- 被传承者：绿色边框
- 传承连线：金色虚线动画
- 可传承项：白色卡片
- 已传承项：灰色半透明

交互效果：
- 选择角色有金光连线
- 传承有能量流动动画
- 传承完成有粒子爆发
- 连线有脉冲流动效果
```
