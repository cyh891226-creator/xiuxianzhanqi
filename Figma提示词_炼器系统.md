# Figma提示词 - 炼器系统

## 模块概述

炼器系统是游戏的核心养成系统之一，通过收集材料、打造装备，为玩家提供强力装备和法宝。炼器界面需要展示炼器炉、图纸、材料和炼器师等信息。

---

## 1. 炼器主界面（Smithing Main Interface）

### 整体布局

```
┌─────────────────────────────────────────────────────────────┐
│  [返回]  炼器房  [炼器师信息]                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────────────────────────────────────────┐      │
│   │                   炼器炉区域                      │      │
│   │                                                 │      │
│   │              ┌───────────────┐                   │      │
│   │              │   炼器炉图标   │                   │      │
│   │              │   (动态火焰)   │                   │      │
│   │              └───────────────┘                   │      │
│   │                                                 │      │
│   │         [选择图纸▼]  [开始打造]                  │      │
│   │                                                 │      │
│   └─────────────────────────────────────────────────┘      │
│                                                             │
│   ┌───────────────┐  ┌───────────────┐                   │
│   │   材料背包     │  │   图纸列表     │                   │
│   │   [图标]       │  │   [图标]       │                   │
│   └───────────────┘  └───────────────┘                   │
│                                                             │
│   ┌─────────────────────────────────────────────────┐      │
│   │                   炼器师信息                       │      │
│   │   [头像] 等级：初级炼器师  成功率+10%           │      │
│   └─────────────────────────────────────────────────┘      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 提示词

```
创建一个修仙战棋游戏的炼器系统主界面设计。

布局要求：
1. 顶部导航栏（高度56px）：
   - 左侧：返回按钮（箭头图标）
   - 中央：炼器房标题（修仙风字体）
   - 右侧：当前炼器师信息（头像+等级）

2. 炼器炉区域（占据主要空间）：
   - 炼器炉图标展示（120x120px，动态火焰效果）
   - 下方：选择图纸下拉框
   - 下方：开始打造按钮

3. 功能面板（左右排列）：
   - 左侧：材料背包按钮（显示材料数量角标）
   - 右侧：图纸列表按钮

4. 底部：炼器师信息卡片
   - 炼器师头像
   - 炼器师等级
   - 成功率加成显示

设计风格：
- 整体背景：炼器房风格，有火焰光效
- 炼器炉：带动态火焰动画，品质越高火焰越绚烂
- 按钮：圆角矩形，炼器相关图标
- 面板：半透明深色背景

交互效果：
- 炼器炉有呼吸光效
- 选择图纸后显示所需材料
- 开始打造有火焰锤击动画
- 打造完成有装备出现动画
```

### 提示词

```
创建一个修仙战棋游戏的装备打造界面设计。

布局要求：
1. 顶部导航栏：
   - 左侧：返回按钮
   - 中央：打造状态（打造中/打造完成）

2. 炼器炉展示区（主要空间）：
   - 大型炼器炉图标（200x200px）
   - 动态锤击动画效果
   - 打造进度条（百分比显示）

3. 材料显示区：
   - 显示所需材料及数量
   - 每个材料显示：图标 + 名称 + 数量
   - 已放入显示对勾标识
   - 缺少材料显示红色标识

4. 打造信息：
   - 预计获得物品
   - 当前成功率百分比
   - 品质提升概率

5. 底部按钮：
   - 取消打造（次要按钮）

设计风格：
- 炼器炉火焰：根据打造进度变化颜色
- 进度条：金色填充，云纹装饰
- 材料：图标+文字组合，绿色对勾/红色叉号

交互效果：
- 打造中有锤击动画
- 进度条平滑增长
- 完成时有装备闪光特效
- 成功/失败有不同动画反馈
```

创建一个修仙战棋游戏的装备强化界面设计。

   - 左侧：返回按钮
