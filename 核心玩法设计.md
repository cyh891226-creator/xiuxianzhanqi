# 天地劫—万世仙宗 - 核心战斗系统设计

## 一、游戏类型定位

### 1.1 游戏类型
- **类型**：回合制策略战棋游戏（SRPG）
- **平台**：PC / 移动端 / 主机
- **风格**：东方玄幻修仙题材
- **核心体验**：策略战斗 + 战术深度 + 团队配合

### 1.2 目标玩家
- 喜欢策略战棋游戏的玩家
- 修仙/武侠文化爱好者
- 追求深度策略和战术思考的玩家
- 喜欢东方玄幻美术风格的玩家
- 喜欢团队协作和战术搭配的玩家

## 二、核心战斗系统

### 2.1 战斗模式

#### 回合制战棋
- **地图规格**：标准地图 15×15 格子，大型地图 20×20 格子
- **参战单位**：玩家队伍 4-6 人 vs 敌方队伍
- **胜利条件**：
  - 歼灭战：消灭所有敌方单位
  - 斩首战：击败敌方首领
  - 生存战：坚持指定回合数
  - 护送战：保护目标到达指定地点
  - 夺旗战：占领指定据点
  - **救援战**：救出被困单位
  - **撤退战**：带领指定单位安全撤离

#### 战斗流程
```
1. 准备阶段 → 2. 部署阶段 → 3. 行动阶段 → 4. 敌方回合 → 5. 结算阶段
```

### 2.2 移动系统

#### 基础移动
- **移动力**：根据角色职业和属性决定，通常为 3-6 格
- **移动方式**：
  - 普通移动：消耗移动力，每格消耗 1 点
  - 飞行移动：无视地形障碍，但消耗更多移动力
  - 瞬移：特殊技能，直接到达指定位置

#### 地形影响
| 地形类型 | 移动消耗 | 效果 |
|----------|----------|------|
| 平地 | 1 | 无特殊效果 |
| 山地 | 2 | 防御+20%，命中+10%，移动力-1 |
| 水域 | 3 | 闪避-10%，部分单位无法通过 |
| 森林 | 2 | 闪避+15%，视野受限，隐藏效果 |
| 沼泽 | 3 | 每回合损失 5% 生命值，移动后可能陷入 |
| 灵脉 | 1 | 每回合恢复 5% 法力值，技能效果+10% |
| 毒瘴 | 2 | 每回合可能中毒，持续伤害 |
| 火域 | 2 | 火属性伤害+30%，可能点燃 |
| 冰面 | 2 | 速度+1，可能滑倒 |
| 沙漠 | 2 | 每回合损失 3% 生命值，法防-10% |

### 2.3 攻击系统

#### 攻击类型
- **普通攻击**：基础物理攻击，不消耗法力
- **法术攻击**：消耗法力，造成元素伤害
- **反击**：被攻击时自动反击

#### 攻击范围
- **近战**：周围 1 格（十字或八方向）
- **远程**：直线或抛物线，距离 2-5 格
- **范围**：指定区域，影响多个目标
- **全屏**：影响战场上所有单位

#### 命中率计算
```
命中率 = 基础命中率 + 攻击方命中 - 防御方闪避 + 地形修正 + 距离修正
最小命中率：5%，最大命中率：95%

距离修正：距离越远，命中率越低
```

### 2.4 防御系统

#### 防御类型
- **物理防御**：减少物理伤害
- **法术防御**：减少法术伤害
- **闪避**：完全躲避攻击的概率
- **格挡**：减少部分伤害并反击的概率
- **抗性**：对特定属性的伤害减免
- **护甲**：吸收固定伤害

#### 伤害计算公式
```
物理伤害 = max(1, (攻击力 - 防御力) × 技能系数 × 暴击系数 × 随机系数 - 护甲)
法术伤害 = max(1, (法术攻击 - 法术防御) × 技能系数 × 属性克制 × 随机系数)

随机系数：0.9 - 1.1 之间随机
```

### 2.5 战斗资源

#### 生命值（HP）
- 归零时角色死亡或撤退
- 可以通过丹药、法术恢复
- **重伤状态**：生命值低于30%时，属性下降

#### 法力值（MP）
- 使用法术和技能消耗
- 每回合自动恢复少量
- 可以通过丹药、灵脉恢复

#### 移动力系统
- **移动力**：根据角色速度属性决定，基础移动力=速度值÷10（向下取整），最小2格，最大10格
- **移动消耗**：每移动1格消耗1点移动力
- **攻击消耗**：第一次攻击/技能免费，不消耗移动力；第二次攻击/技能消耗2点移动力；第三次攻击/技能消耗3点移动力
- **技能消耗**：根据技能等级消耗2-5点移动力
- **多攻击机制**：如果剩余移动力≥攻击消耗，可以再次攻击，每回合最多3次攻击或技能

## 三、战斗策略要素

### 3.1 连携系统

- **连携攻击**：作为被动技能存在，当多个角色攻击同一目标时触发
- **效果**：连携次数越多，伤害加成越高，最多 6 人连携，伤害+100%
- **触发条件**：同一目标在同一回合内受到多个角色攻击

### 3.2 场景道具

- 战场上散布着可互动的道具
- 如：恢复灵泉、爆炸火油、传送阵等
- 合理利用可以扭转战局

## 四、五行相克系统

### 4.1 五行相克关系

```
金 → 木 → 土 → 水 → 火 → 金
（循环相克）
```

### 4.2 克制效果
- **克制方**：伤害+30%，命中率+10%
- **被克制方**：伤害-20%，闪避-10%

---

## 五、战斗平衡设计

### 5.1 职业平衡
- 各职业定位明确，互相补充
- 没有绝对强势的职业，需要团队配合
- 不同职业在不同场景下有优势

### 5.2 技能平衡
- 技能效果多样，注重策略性
- 技能威力差距控制在合理范围内
- 技能冷却和消耗与威力匹配

### 5.3 境界平衡
- 高境界对低境界有合理的压制效果
- 压制效果适度，避免碾压
- 低境界角色通过策略仍有机会战胜高境界对手

## 六、与其他系统的关联

### 6.1 与角色属性系统的关联
- 六维属性（根骨、体魄、悟性、气感、神识、身法）直接影响战斗属性
- 属性潜力影响角色成长和战斗表现
- 境界突破提升角色战斗能力

### 6.2 与技能系统的关联
- 技能通过功法学习和解锁
- 技能槽随境界解锁
- 技能升级和解锁靠角色升级且属性达标
- 技能树只是可学习的技能列表以树状图展示

### 6.3 与血脉系统的关联
- 血脉潜力影响角色属性成长
- 血脉特性提供战斗加成
- 血脉技能在血脉功法中，不与血脉天赋强制关联

### 6.4 与装备系统的关联
- 装备提升角色战斗属性
- 法宝系统提供额外战斗能力
- 装备强化和套装效果增强战斗表现

---

## 七、核心战斗系统总结

核心战斗系统以回合制战棋为基础，融合修仙题材特色，提供以下核心体验：

1. **战术深度**：地形影响、反击机制、疲劳系统、命中率计算，提供丰富的策略选择
2. **团队协作**：连携攻击作为被动技能，强调团队配合
3. **策略多样性**：五行相克、场景道具，增加战斗变数
4. **平衡设计**：职业平衡、技能平衡、境界平衡，确保公平竞技
5. **系统整合**：与角色属性、技能、血脉、装备系统深度融合

这个核心战斗系统设计为游戏提供了坚实的战斗框架，结合修仙题材的特色元素，为玩家带来富有策略性和深度的战棋战斗体验。
