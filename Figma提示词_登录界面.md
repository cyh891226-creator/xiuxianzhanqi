# Figma提示词 - 登录界面

## 模块概述
登录界面是玩家进入游戏的第一印象界面，展示游戏Logo、名称，并提供版本公告、开始游戏、读取存档、成就等功能入口。

### 提示词

```
创建一个修仙题材回合制战棋游戏的登录界面设计。

布局要求：

1. 顶部区域（预留空间，可放广告图）：
   - 高度：120px
   - 可放置游戏宣传图或活动图
   - 如无内容则显示默认云雾动画背景

2. LOGO展示区（高度280px）：
   - 中央位置显示游戏LOGO
   - LOGO尺寸：200x200px
   - LOGO带有动态光效（粒子环绕/光芒脉冲）
   - LOGO下方有游戏名称

3. 游戏名称：
   - 主标题：天地劫
   - 副标题：万世仙宗
   - 字体：古风水墨字体（如有定制字体优先使用）
   - 文字带有微光效果

4. 版本公告面板（可折叠/展开）：
   - 面板尺寸：宽度80%，高度auto
   - 公告标题带图标
   - 公告内容显示2-3行预览
   - "查看详情"按钮
   - 点击可展开完整公告

5. 主功能按钮区（居中排列）：
   - 按钮尺寸：宽度60%，高度56px
   - 按钮间距：16px
   - 按钮样式：圆角矩形，金色边框

   按钮1 - 新的开始：
   - 图标：剑/开始图标
   - 文字：新的开始
   - 样式：主要按钮，金色渐变背景
   - hover效果：发光脉冲

   按钮2 - 读取存档：
   - 图标：文件夹/存档图标
   - 文字：读取存档
   - 样式：次要按钮，白色边框
   - 有存档时显示数量角标

   按钮3 - 成就中心：
   - 图标：奖杯/成就图标
   - 文字：成就中心
   - 样式：次要按钮，白色边框

6. 底部信息栏：
   - 版本号显示
   - 版权信息
   - 高度：40px

设计风格：
- 整体背景：深青灰色渐变背景（#1a1a2e → #0d0d1a）
- 云雾效果：淡金色/白色云雾飘动
- LOGO边框：金色光芒环绕
- 按钮：圆角16px，金色边框+阴影
- 字体：标题使用宋体/楷体，正文使用黑体

配色方案：
- 主色调：深青灰 #1a1a2e
- 强调色：金色 #ffd700
- 文字色：白色 #ffffff
- 次要文字：灰色 #888888
- 按钮背景：渐变金色 #ffd700 → #ff9500

交互效果：
- LOGO有持续的光芒粒子动画
- 云雾背景缓慢飘动
- 按钮hover有光晕扩散效果
- 点击按钮有按压反馈
- 公告面板可点击展开/收起
- 存档数量角标闪烁（如有存档）
```

## 8. 登录界面完整流程

### 流程说明

```
[启动游戏]
     ↓
[Loading界面] → 加载资源、初始化游戏
     ↓
[登录主界面]
     ↓
[存档读取] → 有存档则选择，无则进入创建
     ↓
[创建角色] → 设置名称、性别、职业
     ↓
[剧情过渡] → 播放开场动画
     ↓
[进入游戏]
```

## 设计规范总结

### 通用尺寸
- 弹窗最大宽度：480px
- 按钮高度：48-56px
- 按钮圆角：12-16px
- 内边距：16-24px

### 通用颜色
- 背景色：#1a1a2e（深青灰）
- 强调色：#ffd700（金色）
- 文字色：#ffffff（白色）
- 次要文字：#888888（灰色）
- 成功色：#2ecc71（绿色）
- 警告色：#e74c3c（红色）

### 通用动画
- 弹窗出现：从底部滑入 + 渐变
- 按钮hover：光晕扩散
- 按钮点击：按压缩放
- 进度条：平滑填充
- 过渡时间：300-500ms

### 字体规范
- 标题：24-32px，粗体
- 正文：16px，常规
- 辅助文字：12-14px
- 按钮文字：18px，粗体
