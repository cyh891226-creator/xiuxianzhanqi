# Figma提示词 - 宗门系统

## 模块概述
宗门系统是游戏的核心养成和经营模块，包含宗门建设、设施管理、弟子培养、领地扩张等功能。

---

### 提示词

```
创建一个修仙题材回合制战棋游戏的宗门主页界面。

布局要求：
1. 顶部信息栏（高度64px）：
   - 左侧：返回按钮
   - 中央：宗门名称
   - 右侧：宗门商店按钮

2. 宗门概况卡片（顶部）：
   - 宗门名称 + 等级徽章
   - 宗主名称 + 头像
   - 成员数量：当前/上限
   - 灵石储备 + 声望值

3. 功能入口（5个卡片网格）：
   - 宗门大殿：宗门基础信息
   - 设施升级：建筑升级管理
   - 弟子管理：成员列表和培养
   - 科技研究：宗门科技树
   - 领地扩张：地图和领土

4. 宗门领地地图（中央区域）：
   - 2D俯视图展示宗门区域
   - 主殿在中央位置
   - 周边建筑环绕
   - 领地边界用虚线标记

5. 底部快捷入口：
   - 宗门任务按钮
   - 宗门日志按钮
   - 邀请好友按钮

设计风格：
- 界面：暗色古风背景
- 宗门概况：金色边框卡片
- 功能入口：云纹装饰卡片
- 领地地图：水墨山水风格
- 等级徽章：金色圆形

交互效果：
- 点击建筑有光晕选中效果
- 领地地图可缩放拖拽
- 灵石/声望变化有数字跳动
- 新任务有角标提示
```

## 3. 设施升级界面（Facility Upgrade Interface）

### 提示词

```
创建一个修仙战棋游戏的设施升级界面。

布局要求：
1. 设施分类标签：
   - 全部 | 生产 | 修炼 | 其他

2. 设施列表（可滚动）：
   每种设施卡片：
   - 设施图标（64x64）
   - 设施名称 + 等级
   - 设施效果描述
   - 升级消耗材料列表
   - 升级按钮 / 升级中状态

3. 设施详情弹窗（点击设施后）：
   - 设施完整描述
   - 当前等级效果
   - 下一级效果预览
   - 升级材料需求
   - 升级按钮

4. 特殊设施（需解锁）：
   - 显示为锁定状态
   - 解锁条件说明
   - 解锁进度

5. 批量操作：
   - 一键升级按钮（自动使用材料）
   - 升级队列显示

设计风格：
- 设施卡片：白色圆角卡片
- 已满级：金色边框
- 可升级：蓝色高亮
- 升级中：进度条动画
- 锁定设施：灰色 + 锁图标

交互效果：
- 升级有建造动画
- 材料不足有抖动提示
- 升级完成有金光效果
- 队列有倒计时显示
```
